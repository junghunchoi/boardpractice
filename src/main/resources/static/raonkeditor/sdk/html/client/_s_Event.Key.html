K Editor :: Event.Key

Script : Event.Key

에디터 디자인 영역에서 키보드 동작 시 발생하는 callback 이벤트를 설정합니다.

remarks

paramObj.strEventName은 이벤트 이름, paramObj.objEvent는 이벤트 객체, paramObj.objElement는 이벤트가 발생한 요소를 의미합니다.
에디터의 설정 파일(raonkeditor/config/raonk_editor.xml)의 return_event 설정의 key_event 속성 값이 1인 경우 이벤트가 발생합니다.
