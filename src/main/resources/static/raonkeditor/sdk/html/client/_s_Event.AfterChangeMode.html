K Editor :: Event.AfterChangeMode

Script : Event.AfterChangeMode

status bar의 모드 변경 시 발생하는 callback 이벤트를 설정합니다.

remarks

paramObj.strNewMode는 변경 후 status 모드, paramObj.strBeforeMode는 변경 전 status mode를 의미합니다.
design인 경우 디자인 모드, source인 경우 HTML 모드, preview인 경우 미리보기 모드, text인 경우 TEXT 모드를 의미합니다.
