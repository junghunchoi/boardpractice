K Editor :: UseLineHeightKeyin

XML : <line_height use_inc_dec="">.....</line_height>, Script : UseLineHeightKeyin

Script 설정이 우선 적용 됩니다.
tool bar의 줄간격 콤보박스옆 위아래 클릭도구를 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 tool bar의 줄간격 콤보박스옆 위아래 클릭도구를 사용합니다.

sample code

tool bar의 줄간격 콤보박스옆 위아래 클릭도구를 사용하도록 설정합니다.

XML

.....
<line_height use_inc_dec="1"></line_height>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseLineHeightKeyin: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
