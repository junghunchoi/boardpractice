K Editor :: ImageContinueInsertMaintainValue

XML : <image_continue_insert_maintain_value>.....</image_continue_insert_maintain_value>,
Script : ImageContinueInsertMaintainValue

Script 설정이 우선 적용 됩니다.
이미지 계속 삽입 체크 시 값 유지 체크 여부를 설정합니다.

remarks

0인 경우 값 유지 체크 하지 않음, 1인 경우 값 유지 체크 함을 의미합니다.
값 유지 체크 시 너비, 높이, ID를 제외한 값이 유지됩니다.

sample code

값 유지에 체크합니다.

XML

.....
<setting>
    .....
    <image_continue_insert_maintain_value>1</image_continue_insert_maintain_value>
    .....
</setting>
.....
