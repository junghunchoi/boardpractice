[관리자 라이선스 전용]

K Editor :: EventList

XML : <event_list></event_list>, Script : EventList

Script 설정이 우선 적용 됩니다.
Form 요소들에 대하여 기본적으로 추가하고자 하는 이벤트명을 설정합니다.

remarks

Script 설정이 두개 이상인 경우 ,(쉼표)로 구분합니다.

sample code

Form 태그에 적용할 수 있는 이벤트를 설정합니다.

XML

.....
<setting>
    .....
    <manager_mode use="1">
        <event_list>
            <event>onblur</event>
            <event>onchange</event>
            <event>onsubmit</event>
            <event>onkeydown</event>
            <event>onkeyup</event>
            <event>onkeypress</event>
            <event>onmousemove</event>
            <event>onmousedown</event>
            <event>onmouseup</event>
            <event>onclick</event>
            <event>ondblclick</event>
        </event_list>
    </manager_mode>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            EventList: 'onchange,onsubmit,onkeydown,onkeyup,onkeypress,onmousemove,onmousedown,onmouseup,onclick,ondblclick'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
