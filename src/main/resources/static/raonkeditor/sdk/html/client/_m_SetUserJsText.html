K Editor :: SetUserJsText

void SetUserJsText(jsText, editorName)

에디터 작성 영역에 사용자의 JS를 적용합니다.

return value

없음.

parameters

jsText  사용자가 적용할 JS를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.
