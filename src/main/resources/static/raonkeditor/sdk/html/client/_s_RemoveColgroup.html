K Editor :: RemoveColgroup

XML : <remove_colgroup>.....</remove_colgroup>, Script : RemoveColgroup

Script 설정이 우선 적용 됩니다.
붙여넣기 시 테이블의 colgroup을 유지/삭제 하도록 설정합니다.

remarks

기본값은 "1" 이며 붙여넣기 시 테이블의 colgroup을 삭제하며,
"0"으로 설정시 삭제하지 않습니다.

sample code

붙여넣기 시 테이블의 colgroup을 유지하도록 설정합니다.

XML

.....
<setting>
    .....
    <remove_colgroup>0</remove_colgroup>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            RemoveColgroup: '0'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
