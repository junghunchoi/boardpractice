K Editor :: ImgDefaultSize

XML : <img_default_size width="">.....</img_default_size>, Config : config.ImgDefaultWidth
XML : <img_default_size height="">.....</img_default_size>, Config : config.ImgDefaultHeight

이미지 다이얼로그창의 너비, 높이 기본값을 설정합니다.

remarks

예) width="100" 설정 시
너비는 "100" / 높이는 너비의 값에 비례하여 적용됩니다.
(실제 이미지 사이즈가 설정값 보다 클 경우에 적용됩니다.)

width 속성값을 % 로 설정 가능합니다.
% 설정시 이미지 다이얼로그 팝업의 이미지 사이즈 Unit 기본값이 %로 설정됩니다.
"%"로 설정 시 이미지 다이얼로그에 이미지 추가 시 width값을 에디터 작성영역 기준 원본 크기의 %로 설정합니다.
값을 포함한 %로 설정 시 [예) 100%] 에디터 작성영역에 대한 해당 이미지 % 비율이 설정값을 초과할 경우 해당 설정값으로 적용됩니다.
(실제 이미지 사이즈가 설정값 보다 클 경우에 적용됩니다.)

sample code

이미지 다이얼로그에 이미지 추가 시 width값을 에디터 작성영역 기준 이미지 원본 크기의 %로 설정합니다.

    <img_default_size width="%" height=""></img_default_size>
    .....
