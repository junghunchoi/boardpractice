K Editor :: GetWrittenAreaSize

object GetWrittenAreaSize(editorName)

생성된 에디터의 작성영역 및 작성영역에 작성된 크기(width, height)를 반환해줍니다.

return value

현재 생성되어있는 에디터의 작성영역(body)의 사이즈(min_width, min_height) 및 작성된 영역의 사이즈(width, height)
※ 작성된 영역의 사이즈는 생성된 에디터의 작성영역(body)의 사이즈보다 작을 수 없습니다.

parameters

editorName  API를 적용할 에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">
        function writtenAreaSize(){
            // id가 editor1인 에디터의 작성영역 및 작성영역에 작성된 크기(width, height)를 반환받습니다.
            var sizeObj = RAONKEDITOR.getWrittenAreaSize('editor1');
        
            console.log("editorBodyWidth = " + sizeObj.min_width); // 현재 생성되어있는 에디터의 작성영역(body)의 width
            console.log("editorBodyHeight = " + sizeObj.min_height); // 현재 생성되어있는 에디터의 작성영역(body)의 height
            console.log("writtenAreaWidth = " + sizeObj.width); // 작성된 영역의 width (min_width 보다 작을 수 없습니다.)
            console.log("writtenAreaHeight = " + sizeObj.height); // 작성된 영역의 height (min_height 보다 작을 수 없습니다.)
        }
</script>
