K Editor :: AddHttpHeader

void AddHttpHeader(headerName, headerValue, editorName)

에디터에 파일 업로드 시 (이미지, 비디오 태그, 영상, 파일 등) 헤더를 추가합니다.

return value

없음.

parameters

headerName  헤더의 이름을 의미합니다.
headerValue  헤더의 값을 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.
