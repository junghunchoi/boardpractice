K Editor :: DecodeMime

string DecodeMime(paramObj, editor_id)

mimeData를 decode 합니다.

return value

정상 동작 시 decode 된 string과, 에디터 id를 리턴하고,
오류 시 "0"과, 에디터 id를 리턴합니다.

parameters

paramObj.mimeData  디코딩할 mimeData 입니다.
paramObj.callback  비동기 callback으로 리턴합니다.
editor_id  에디터 Id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
<script type="text/javascript">

    function decodeMime() {
        // mimeData를 decode 한 후 callback function으로 리턴받습니다.
           
        var callback = function(result){
            // 비동기 콜백으로 리턴됩니다. 
            console.log('editorID : ' + result.strEditorId);
            console.log('strData : ' + result.strData);
        }

        var mimedata = "decode 할 mimeData";
        var paramObject = {
            mimedata : mimedata,
            callback : callback
        }
        RAONKEDITOR.DecodeMime(paramObject, 'editor1');
    }

</script>
