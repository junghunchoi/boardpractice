K Editor :: AddHtmlToSetValue

void AddHtmlToSetValue(html, preOrSub, editorName)

에디터에 set 하는 API를 사용할때 내용의 앞 또는 뒤에 파라미터의 html 소스를 추가하도록 설정합니다.

return value

없음.

parameters

html  삽입할 HTML 소스를 의미합니다.
preOrSub  HTML 소스를 삽입할 위치를 의미합니다. 추가할 위치를 의미합니다. (0 : 앞, 1 : 뒤)
editorName  에디터의 ID를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function addHtmlToSetValue() {
        // id가 editor1인 에디터가 set API를 사용하면 입력하는 소스의 앞쪽에 html의 내용을 추가합니다.
        var html = '<p>Insert HTML</p>';

        RAONKEDITOR.addHtmlToSetValue(html, 0, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
