K Editor :: AdjustCurrentColorInSetApi

XML : <adjust_currentcolor_in_set_api>.....</adjust_currentcolor_in_set_api>,
Script : AdjustCurrentColorInSetApi

Script 설정이 우선 적용 됩니다.
set API 사용시 태그의 style중 border에 currentcolor 값을 실제 색상값으로 변경해 주도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 set 하는 내용의 태그 style중 border에 currentcolor 값을 실제 색상값으로 변경 합니다.

sample code

set 하는 내용의 태그 style중 border에 currentcolor 값을 실제 색상값으로 변경 합니다.

XML

.....
<setting>
    .....
    <adjust_currentcolor_in_set_api>1</adjust_currentcolor_in_set_api>
    .....
</setting>
.....
