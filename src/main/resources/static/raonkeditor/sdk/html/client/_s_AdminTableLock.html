K Editor :: AdminTableLock

XML : <table_lock default_show_lock_icon_user_mode=""></table_lock>, Script : AdminTableLock.DefaultShowLockIconUserMode
XML : <table_lock default_lock_name=""></table_lock>, Script : AdminTableLock.DefaultLockName
XML : <table_lock check_lock_name=""></table_lock>, Script : AdminTableLock.CheckLockName
XML : <table_lock default_table_lock_mode=""></table_lock>, Script : AdminTableLock.DefaultTableLockMode

Script 설정이 우선 적용 됩니다.
관리자모드에 관련된 설정 중 표 잠금 설정을 적용합니다.

remarks

XML : default_show_lock_icon_user_mode, Script : Use  테이블, 셀의 잠금 여부를 사용자에게 Show(보이기) 또는 Hidden(감추기) 여부를 설정합니다.
                                            ("0"=Hidden, "1"="Show")

XML : default_lock_name, Script : Mode  TABLE/TD/TH에 해당 설정값으로 setAttribute된 테이블 또는 셀을 잠급니다. ('설정값', '')
XML : check_lock_name, Script : CheckLockName  타사 에디터와의 호환성을 위한 값으로 타사의 에디터로 저장된 테이블/셀 잠금 기능을 사용허용합니다.
XML : default_table_lock_mode, Script : DefaultTableLockMode  기본값은 "2"이며 Resize 및 편집제한, 값이 "1"인 경우 테이블에 대한 Resize만 제한합니다.

sample code

관리자모드에 관련된 설정 중 표 잠금 설정을 적용합니다.
