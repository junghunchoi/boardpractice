K Editor :: InsertTextarea

void InsertTextarea(attributesObj, textareaObj, editorName)

에디터 디자인 영역에 <textarea> 태그를 삽입합니다.

return value

없음.

parameters

attributesObj.name  textarea 태그의 name을 의미합니다.
attributesObj.id  textarea 태그의 id를 의미합니다.
attributesObj.class  textarea 태그의 class를 의미합니다.
attributesObj.title  textarea 태그의 title을 의미합니다.
attributesObj.value  textarea 태그의 value를 의미합니다.
attributesObj.rows  textarea 태그의 rows를 의미합니다.
attributesObj.cols  textarea 태그의 cols를 의미합니다.
attributesObj.disabled  textarea 태그의 disabled를 의미합니다.
attributesObj.readonly  textarea 태그의 readonly를 의미합니다.
attributesObj.style.width  textarea 태그의 너비를 의미합니다.
attributesObj.style.height  높이를 의미합니다.
attributesObj.style.text_align  텍스트 정렬 방향을 의미합니다.
textareaObj  textarea 태그 객체를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

textareaObj가 null인 경우 textarea 태그 생성, null이 아닌 경우 기존 textarea 태그를 변경합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function insertTextarea() {
        var attributesObj = {
            name : 'newName',
            id : 'newId',
            class : 'newClass',
            title : 'newTitle',
            value : 'newValue',
            rows : 90,
            cols : 55,
            disabled : false,
            readonly : false,
            'style.width' : '900',
            'style.height' : '550',
            'style.text_align' : 'left'
        };

        // id가 'textareaId'인 textarea 태그
        var _textarea = RAONKEDITOR.getElementById('textareaId', 'editor1');

        RAONKEDITOR.insertTextarea(attributesObj, _textarea, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
