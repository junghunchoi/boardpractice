K Editor :: SetCaretBeforeOrAfter

void SetCaretBeforeOrAfter(option, editorName)

에디터 작성 영역에 있는 노드의 전후로 커서를 이동합니다.

return value

없음.

parameters

option.node  노드를 의미합니다.
option.pos  이동할 위치를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

option.pos가 before일 경우 노드의 앞, after일 경우 노드의 뒤를 의미합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function setCaretBeforeOrAfter() {
        // id가 'elementId'인 요소
        var element = RAONKEDITOR.getElementById('elementId', 'editor1');

        // id가 'elementId'인 요소의 전으로 커서를 이동합니다.
        var style = RAONKEDITOR.setCaretBeforeOrAfter({node : element, pos : 'before'}, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
