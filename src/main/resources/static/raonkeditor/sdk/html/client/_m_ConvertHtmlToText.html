K Editor :: ConvertHtmlToText

string ConvertHtmlToText(strHtml, editorName)

html을 text로 변환 후 p태그로 감싼 값을 반환합니다.

return value

p 태그로 감싸진 html의 text string 이 리턴됩니다.

parameters

strHtml  변경할 html 소스를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.

sample code

        var strHtml = 'Convert HTML to TEXT';

    }
