K Editor :: DisplayFontFamilyList

XML : <display_font_family>.....</display_font_family>, Script : DisplayFontFamilyList

Script 설정이 우선 적용 됩니다.
글꼴 메뉴의 글꼴 이름을 변경합니다.

remarks

기존 글꼴 이름|변경할 글꼴 이름 으로 설정합니다.
Script 설정 시 두개 이상인 경우 ,(쉼표)로 구분합니다.

sample code

'굴림'을 '굴림1', '궁서'를 '궁서1'로 변경합니다.

XML

.....
<display_font_family>
    .....
    <font>굴림|굴림1</font>
    <font>궁서|궁서1</font>
    .....
</display_font_family>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            DisplayFontFamilyList: '굴림|굴림1,궁서|궁서1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
