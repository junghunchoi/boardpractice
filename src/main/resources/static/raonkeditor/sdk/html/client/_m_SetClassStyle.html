K Editor :: SetClassStyle

void SetClassStyle(className, editorName)

에디터 디자인 영역에서 선택 영역에 태그 생성 후, 태그의 class 이름을 설정합니다.

return value

없음.

parameters

className  태그의 class 이름을 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function setClassStyle() {
        var className = 'elementClass';

        RAONKEDITOR.setClassStyle(className, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
