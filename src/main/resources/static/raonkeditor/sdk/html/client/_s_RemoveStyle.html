K Editor :: RemoveStyle

XML : <use_remove_style>.....</use_remove_style>, Script : RemoveStyle.Use
XML : <use_remove_style tag="">.....</use_remove_style>, Script : RemoveStyle.Tag
XML : <use_remove_style style="">.....</use_remove_style>, Script : RemoveStyle.Style

Script 설정이 우선 적용 됩니다.
set API 사용시 태그의 스타일 삭제여부 및 삭제 조건을 설정합니다.

remarks

XML : use_remove_style, Script : Use  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 set API 사용시 태그의 스타일을 삭제 하도록 설정합니다.
XML : tag, Script : Tag  기본값은 "" 이며 사용안함, style 항목 입력 시 해당 style 항목을 삭제 합니다.
XML : style, Script : Style  기본값은 "" 이며 모든 태그에 적용함, 태그명 입력시 해당 태그의 style만 삭제 합니다.
