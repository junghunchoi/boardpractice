K Editor :: AutoGrowMode

XML : <auto_grow_mode>.....</auto_grow_mode>, Script : AutoGrowMode

Script 설정이 우선 적용 됩니다.
붙여넣기시 에디터 영역의 크기 초과시 자동으로 확장되는 AutoGrowMode 활성화 여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 AutoGrowMode를 사용합니다.

sample code

AutoGrowMode를 사용합니다.

XML

.....
<plugin_setting>
    .....
    <auto_grow_mode>1</auto_grow_mode>
    .....
</plugin_setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            AutoGrowMode: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
