K Editor :: UseReplaceImage

XML : <use_replace_image>.....</use_replace_image>, Script : UseReplaceImage

Script 설정이 우선 적용 됩니다.
작성중 iframe, flash 등에 대체이미지를 사용하는 기능의 활성화 여부를 설정합니다.

remarks

기본값은 "1"로 작성중 iframe, flash 등에 대체이미지를 사용,
"0"으로 설정 시 작성중 iframe, flash 등에 대체이미지를 사용하지 않습니다.

sample code

작성중 iframe, flash 등에 대체이미지를 사용하지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <use_replace_image>0</use_replace_image>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseReplaceImage: '0'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
