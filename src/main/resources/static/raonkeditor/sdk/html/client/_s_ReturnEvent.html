K Editor :: ReturnEvent

XML : <return_event mouse_event="">.....</return_event>, Script : ReturnEventMouse
XML : <return_event keyboard_event="">.....</return_event>, Script : ReturnEventKeyboard
XML : <return_event command_event="">.....</return_event>, Script : ReturnEventCommand
XML : <return_event focus_event="">.....</return_event>, Script : ReturnEventFocus
XML : <return_event drag_event="">.....</return_event>, Script : ReturnEventDrag

Script 설정이 우선 적용 됩니다.
에디터의 이벤트들의 활성화/비활성화 여부를 설정합니다.

remarks

아래 모든 설정값의 기본값은 "0" 이며 설정값이 "0"일 경우 해당 이벤트 발생하지 않습니다. XML : mouse_event, Script : ReturnEventMouse  "1"로 설정시 RAONKEDITOR_MouseEvent를 사용 가능하도록 설정합니다.
XML : keyboard_event, Script : ReturnEventKeyboard  "1"로 설정시 RAONKEDITOR_KeyEvent를 사용 가능하도록 설정합니다.
XML : command_event, Script : ReturnEventCommand  "1"로 설정시 RAONKEDITOR_CommandEvent를 사용 가능하도록 설정합니다.
XML : focus_event, Script : ReturnEventFocus  "1"로 설정시 RAONKEDITOR_FocusEvent를 사용 가능하도록 설정합니다.
XML : drag_event, Script : ReturnEventDrag  "1"로 설정시 RAONKEDITOR_DragEvent를 사용 가능하도록 설정합니다.

sample code

에디터의 이벤트들을 사용합니다.

XML

.....
<setting>
    .....
    <return_event mouse_event="1" keyboard_event="1" command_event="1" focus_event="1" drag_event="1"></return_event>
    .....
<setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

            ReturnEventCommand: '1',

        }

        new RAONKEditor(raonkParam);
    </script>
