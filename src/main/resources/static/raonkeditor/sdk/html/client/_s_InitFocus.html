K Editor :: InitFocus

XML : <init_focus>.....</init_focus>, Script : InitFocus

Script 설정이 우선 적용 됩니다.
에디터 로드시 에디터 작성영역에 대한 focus의 여부를 설정합니다.

remarks

기본값은 "0" 이며 에디터 로드시 에디터 작성영역에 foucs를 부여하지 않음,
"1" 로 설정 시 에디터 로드시 에디터 작성영역에 foucs를 부여합니다.

sample code

에디터 로드시 에디터 작성영역에 foucs를 부여하지 않도록 합니다.

XML

.....
<setting>
    .....
    <init_focus>0</init_focus>
    .....
</setting>
.....
