K Editor :: AddUserJsUrl

void AddUserJsUrl(url, editorName)

에디터 작성 영역에 사용자의 JS 파일을 적용합니다.

return value

없음.

parameters

url  JS 파일 URL을 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

에디터의 설정 파일(raonkeditor/config/raonk_editor.xml)의 xss_use 설정 값이 1이고,
xss_allow_url 설정의 allow_url에 JS 파일 URL을 설정해야 됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function RAONKEDITOR_CreationComplete() {
        var url = 'http://www.raonk.com/js/userJs.js';

        RAONKEDITOR.addUserJsUrl(url, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
