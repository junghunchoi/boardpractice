K Editor :: AllowFlashFileType

XML : <allow_flash_file_type>.....</allow_flash_file_type>, Script : AllowFlashFileType
XML : <allow_flash_file_type max_file_size="....."></allow_flash_file_type>, Script : MaxFlashFileSize

Script 설정이 우선 적용됩니다.
플래시 삽입 시 허용할 확장자를 설정합니다.
플래시 삽입 시 허용할 최대 파일 사이즈를 제한합니다.

remarks

XML : allow_flash_file_type, Script : AllowFlashFileType   확장자를 ","(콤마)로 구분하여 입력합니다.
XML : max_file_size, Script : MaxFlashFileSize   용량을 바이트단위까지 입력합니다.

sample code

플래시 삽입 시 허용할 확장자를 "swf"로 설정합니다.
플래시 삽입 시 허용할 최대 파일 사이즈를 "10MB"로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <allow_flash_file_type max_file_size="10MB">swf</allow_flash_file_type>
    .....
</uploader_setting>
