K Editor :: InsertImg

void InsertImg(attributesObj, divObj, editorName)

에디터 디자인 영역에 이미지를 삽입합니다.

return value

없음.

parameters

attributesObj.name  이미지의 name을 의미합니다.
attributesObj.id  이미지의 id를 의미합니다.
attributesObj.class  이미지의 class를 의미합니다.
attributesObj.title  이미지의 title을 의미합니다.
attributesObj.src  이미지의 src를 의미합니다.
attributesObj.alt  이미지의 alt를 의미합니다.
attributesObj.style.width  이미지의 너비를 의미합니다.
attributesObj.style.height  높이를 의미합니다.
attributesObj.style.text_align  텍스트 정렬 방향을 의미합니다.
imgObj  이미지 객체를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

imgObj가 null인 경우 이미지 삽입, null이 아닌 경우 기존 이미지를 변경합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function insertImg() {
        var attributesObj = {
            name : 'newName',
            id : 'newId',
            class : 'newClass',
            title : 'newTitle',
            src : 'http://www.raonk.com/image/newImage.jpg',
            alt : 'newAlt',
            'style.width' : '900',
            'style.height' : '550',
            'style.text_align' : 'left'
        };

        // id가 'imgId'인 이미지
        var _img = RAONKEDITOR.getElementById('imgId', 'editor1');

        RAONKEDITOR.insertImg(attributesObj, _img, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
