K Editor :: ImageAutoConvert

XML : <image_auto_convert>.....</image_auto_convert>, Script : ImageAutoConvert

Script 설정이 우선 적용 됩니다.
붙여넣기 또는 드래그앤 드랍으로 이미지 삽입 시 설정된 width, height 값에 맞춰 이미지 크기의 비율 조정여부를 설정합니다.

remarks

기본값은 "0" 이며 이미지 크기비율을 원본 비율대로 화면에 표시하고,
"1" 로 설정하는 경우, 붙여넣기 또는 드래그앤 드랍으로 이미지 삽입 시 설정된 width, height 값에 맞춰서 이미지의 크기 비율을 조정합니다.
width, height 값은 ImageConvertHeight, ImageConvertWidth에 설정된 값으로 결정됩니다.

sample code
