K Editor :: EditorBodyEditableMode

XML : <editor_body_editable mode=""></editor_body_editable>, Script : EditorBodyEditableMode

Script 설정이 우선 적용 됩니다.
디자인 영역의 작성 가능 여부가 불가능일 시 키 이벤트, 마우스 이벤트 발생 여부를 설정합니다.

remarks

1인 경우 키 이벤트 발생 안 함, 2인 경우 키 이벤트 및 마우스 이벤트 발생 안 함을 의미합니다.

sample code

키 이벤트 및 마우스 이벤트가 발생하지 않습니다.

XML

.....
<setting>
    .....
    <editor_body_editable mode="2">0</editor_body_editable>
    .....
</setting>
.....
