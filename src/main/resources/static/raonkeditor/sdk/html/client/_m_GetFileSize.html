K Editor :: GetFileSize

void GetFileSize(fileSizePath, callBackFunc, editorName)

파일 크기를 가지고 옵니다.

return value

callback 함수로 파일 크기가 리턴 됩니다.

parameters

fileSizePath  파일 URL을 의미합니다.
callBackFunc  파일 크기를 리턴 받을 callback 함수를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function getFileSize() {
        var fileSizePath = 'http://www.raonk.com/file/textFile.txt';

        var callBackFunc = function(fileSize) {
            alert('파일 크기 : ' + fileSize);
        }

        RAONKEDITOR.getFileSize(fileSizePath, callBackFunc, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->

    </script>
</div>
