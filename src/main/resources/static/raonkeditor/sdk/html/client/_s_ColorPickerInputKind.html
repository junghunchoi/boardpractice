K Editor :: ColorPickerInputKind

XML : <color_picker_input_kind>.....</color_picker_input_kind>, Script : ColorPickerInputKind

Script 설정이 우선 적용 됩니다.
Color Picker의 색상 입력 방식을 설정합니다.

remarks

기본값은 "0" 이고 HEX 값을 입력 방식으로 설정,
"1"로 설정 시 RGB 값을 입력 방식으로 설정합니다.

sample code

Color Picker의 색상 입력 방식을 RGB값으로 설정합니다.

XML

.....
<setting>
    .....
    <color_picker_input_kind>1</color_picker_input_kind>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->
