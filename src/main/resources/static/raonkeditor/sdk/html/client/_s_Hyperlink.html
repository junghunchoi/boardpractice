K Editor :: Hyperlink

XML : <target>.....</target>, Script : HyperlinkTarget
XML : <category>.....</category>, Script : HyperLinkCategoryList
XML : <protocol>.....</protocol>, Script : HyperLinkProtocolList
XML : <click_ctrl_hyperlink>.....</click_ctrl_hyperlink>, Script : ClickCtrlHyperlink

Script 설정이 우선 적용 됩니다.
하이퍼링크 삽입 다이얼로그 및 하이퍼링크 속성을 설정합니다.

remarks

XML : target, Script : HyperlinkTarget  하이퍼링크 팝업 창의 대상 프레임 목록을 설정합니다. blank, self, top, parent가 있습니다. 두개 이상인 경우 ,(쉼표)로 구분합니다.
XML : category, Script : HyperLinkCategoryList  하이퍼링크 category를 콤마(,)로 구분하여 설정합니다. (hyperlink,bookmark,image 등)
XML : protocol, Script : HyperLinkProtocolList  하이퍼링크 클릭시 프로토콜을 콤마(,)로 구분하여 설정합니다.(http://,https://,file:///,ftp://,gopher://,telnet://,wais://,mailto:,news: 등)
XML : click_ctrl_hyperlink, Script : ClickCtrlHyperlink  기본값은 "0" 이고 사용 안함, "1"로 설정 시 디자인 모드에서 하이퍼링크를 Ctrl + 클릭하면 바로 이동하도록 설정합니다.

sample code

하이퍼링크 삽입 다이얼로그 및 하이퍼링크 속성을 설정합니다.

XML

.....
<hyperlink_property>
    .....
    <target>blank</target>
    <category>hyperlink,bookmark,image</category>
    <protocol>http://,https://,file:///,ftp://,gopher://,telnet://,wais://,mailto:,news:</protocol>
    <click_ctrl_hyperlink>1</click_ctrl_hyperlink>
    .....
<hyperlink_property>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            HyperlinkTarget: 'blank',
            HyperLinkCategoryList: 'hyperlink,bookmark,image',
            HyperLinkProtocolList: 'http://,https://,file:///,ftp://,gopher://,telnet://,wais://,mailto:,news:',
            ClickCtrlHyperlink: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
