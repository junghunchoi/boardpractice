Script 설정이 우선 적용 됩니다.
Get API 호출 시 태그가 비어 있는 경우 &nbsp;로 대체 여부를 설정합니다.

remarks

0인 경우 태그가 비어 있는 경우 &nbsp;로 대체하지 않음, 1인 경우 태그가 비어 있는 경우 &nbsp;로 대체함을 의미합니다.
