K Editor :: UseTablePasteDialog

XML : <use_table_paste_dialog>.....</use_table_paste_dialog>, Script : UseTablePasteDialog

Script 설정이 우선 적용 됩니다.
표 안에 표 붙여넣기 시 표 붙여넣기 팝업 창 사용 여부를 설정합니다.

remarks

0인 경우 표 붙여넣기 팝업 창 사용하지 않음, 1인 경우 표 붙여넣기 팝업 창 사용함을 의미합니다.
표 붙여넣기 팝업 창 사용 시
표 붙여넣기, 표 내용 붙여넣기, 오른쪽에 세로줄로 붙여넣기, 왼쪽에 세로줄로 붙여넣기, 위쪽에 가로줄로 붙여넣기, 아래쪽에 가로줄로 붙여넣기
옵션을 선택할 수 있습니다.

sample code

표 붙여넣기 팝업 창을 사용합니다.

XML

.....
<setting>
    .....
    <use_table_paste_dialog>1</use_table_paste_dialog>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseTablePasteDialog: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
