K Editor :: RemoveEmptyTag

XML : <remove_empty_tag>.....</remove_empty_tag>, Script : RemoveEmptyTag
XML : <remove_empty_tag use_set_value="">.....</remove_empty_tag>, Script : RemoveEmptyTagSetValue
XML : <remove_empty_tag insert_nbsp_for_line_break="">.....</remove_empty_tag>, Script : RemoveEmptyTagInsertNbspForLineBreak

Script 설정이 우선 적용 됩니다.
get API 사용시 내용이 없는 빈 태그 삭제 여부을 적용합니다.
설정값에 따라서 set API 사용시에도 적용합니다.

remarks

XML : remove_empty_tag, Script : RemoveEmptyTag  기본값은 "1"이며 get API 사용시 빈 태그 삭제함, "0"으로 설정시 삭제하지 않습니다.
XML : use_set_value, Script : RemoveEmptyTagSetValue  RemoveEmptyTag 옵션을 set API 사용시에도 적용할지에 대한 옵션을 설정합니다.
XML : insert_nbsp_for_line_break, Script : RemoveEmptyTagInsertNbspForLineBreak  기본값은 "1"이며 RemoveEmptyTag 옵션을 사용 시 삭제 대신 태그안에 &nbsp;를 넣어주도록 설정합니다.

sample code

빈 태그 삭제 관련 설정들을 적용합니다.

    <remove_empty_tag use_set_value="1" insert_nbsp_for_line_break="1">1</remove_empty_tag>
    .....
