K Editor :: UseNoncreationAreaBackground

XML : <auto_body_fit use_noncreation_area_background="">.....</auto_body_fit>,
Script : UseNoncreationAreaBackground

Script 설정이 우선 적용 됩니다.
비 작성영역에 배경색 적용하도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 비 작성영역에도 배경색을 적용합니다.

sample code

비 작성영역에도 배경색을 적용합니다.

XML

.....
<setting>
    .....
    <auto_body_fit use_noncreation_area_background="1"></auto_body_fit>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseNoncreationAreaBackground: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
