K Editor :: InsertDiv

void InsertDiv(attributesObj, divObj, editorName)

에디터 디자인 영역에 <div> 태그를 삽입합니다.

return value

없음.

parameters

attributesObj.name  div 태그의 name을 의미합니다.
attributesObj.id  div 태그의 id를 의미합니다.
attributesObj.class  div 태그의 class를 의미합니다.
attributesObj.title  div 태그의 title을 의미합니다.
attributesObj.alt  div 태그의 alt를 의미합니다.
attributesObj.style.width  div 태그의 너비를 의미합니다.
attributesObj.style.height  높이를 의미합니다.
attributesObj.style.text_align  텍스트 정렬 방향을 의미합니다.
divObj  div 태그 객체를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

divObj가 null인 경우 div 태그 생성, null이 아닌 경우 기존 div 태그를 변경합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function insertDiv() {
        var attributesObj = {
            name : 'newName',
            id : 'newId',
            class : 'newClass',
            title : 'newTitle',
            alt : 'newAlt',
            'style.width' : '900',
            'style.height' : '550',
            'style.text_align' : 'left'
        };

        // id가 'divId'인 div 태그
        var _div = RAONKEDITOR.getElementById('divId', 'editor1');

        RAONKEDITOR.insertDiv(attributesObj, _div, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
