K Editor :: Event.Command

Script : Event.Command

에디터 메뉴 클릭 및 실행 시 발생하는 callback 이벤트를 설정합니다.

remarks

cmd는 이벤트가 발생한 에디터 메뉴를 의미합니다.
tool은 이벤트가 발생한 시점을 의미합니다. true인 경우 에디터의 메뉴 클릭 시, false인 경우 에디터의 메뉴 실행 시를 의미합니다.
에디터의 설정 파일(raonkeditor/config/raonk_editor.xml)의 return_event 설정의 command_event 속성 값이 1인 경우 이벤트가 발생합니다.
