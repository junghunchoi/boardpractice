K Editor :: ReplaceRgbToHex

XML : <replace_RGB_to_hex>.....</replace_RGB_to_hex>, Script : ReplaceRgbToHex

Script 설정이 우선 적용 됩니다.
get API 사용시 style의 rgb 값을 hex 값으로 변경하도록 설정합니다.

remarks

기본값은 "0"으로 사용 안함,
"1"로 설정 시 get API 사용시 style의 rgb 값을 hex 값으로 변경합니다.

sample code

get API 사용시 style의 rgb 값을 hex 값으로 변경합니다.

XML

.....
<setting>
    .....
    <replace_RGB_to_hex>1</replace_RGB_to_hex>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
