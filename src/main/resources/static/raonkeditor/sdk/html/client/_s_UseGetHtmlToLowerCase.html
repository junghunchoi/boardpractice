[IE 8 이하 전용]

K Editor :: UseGetHtmlToLowerCase

XML : <use_get_htmltolowercase>.....</use_get_htmltolowercase>, Script : UseGetHtmlToLowerCase

Script 설정이 우선 적용 됩니다.
get API 사용 시 대문자인 tag명을 모두 소문자로 변경해주도록 설정합니다.

remarks

"1"로 설정 시 get API 사용 시 대문자인 tag명을 모두 소문자로 변경, "0"으로 설정 시 변경하지 않습니다.
기본값은 "1" 입니다.

sample code

버전 8이하의 IE브라우저에서 get API 사용 시 대문자인 tag명을 모두 소문자로 변경합니다.

XML

.....
<setting>
    .....
    <use_get_htmltolowercase>1</use_get_htmltolowercase>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseGetHtmlToLowerCase: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
