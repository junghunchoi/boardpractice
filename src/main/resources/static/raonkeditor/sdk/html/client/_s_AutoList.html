K Editor :: AutoList

XML : <auto_list use="">.....</auto_list>, Script : AutoList.Use

Script 설정이 우선 적용 됩니다.
글 작성시 자동글머리 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 으로 설정 시 글 작성시 자동글머리 기능을 사용합니다.

sample code

자동글머리 기능을 사용합니다.

XML

.....
<setting>
    .....
    <auto_list use="1"></auto_list>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">
    <script type="text/javascript">
        var raonkParam = {
            .....
            AutoList: {
                Use: '1'
            }
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
