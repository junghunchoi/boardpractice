K Editor :: DisableInsertImage

XML : <disable_insert_image>.....</disable_insert_image>, Script : DisableInsertImage

Script 설정이 우선 적용 됩니다.
모든 이미지를 삽입(업로드)하는 기능을 제한하도록 설정합니다.
두개 이상인 경우 ,(쉼표)로 구분합니다.

remarks

"all" 로 설정 시 모든 이미지를 삽입(업로드)하는 기능을 제한합니다.
문서 배경 이미지, 하이퍼링크, 이미지 삽입, 붙여넣기, 표 배경, 셀 배경, 비디오 섬네일에 적용됩니다.

doc_bg_image : 문서 배경 이미지
hyperlink : 하이퍼링크
image : 이미지 삽입
paste_image : 붙여넣기
table : 표 배경
cell : 셀 배경
video_poster : 비디오 섬네일

sample code

문서 배경 이미지, 하이퍼링크의 이미지 삽입을 비활성화합니다.

XML

.....
<setting>
    .....
    <disable_insert_image>doc_bg_image,hyperlink</disable_insert_image>
    .....
</setting>
.....
