K Editor :: ConvertMMToPxUnit

string ConvertMMToPxUnit(html)

HTML 소스의 mm 단위를 px 단위로 변환합니다.

return value

px 단위로 변환된 HTML 소스가 리턴 됩니다.

parameters

html  HTML 소스를 의미합니다.

remarks

없음.
