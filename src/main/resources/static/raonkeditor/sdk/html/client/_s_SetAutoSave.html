Script 설정이 우선 적용 됩니다.
에디터에서 자동저장에 관련된 기능을 설정합니다.

remarks

XML : set_auto_save, Script : Mode  기본값은 "2" 이고 에디터 환경설정에서 사용자가 설정, "0"으로 설정 시 사용 안함, "1"로 설정 시 자동저장 기능을 사용합니다.
XML : interval, Script : Interval  자동저장 주기를 숫자만 입력하여 설정합니다. (단위는 "분" 입니다.)
XML : max_count, Script : MaxCount  자동저장될 최대 개수를 숫자만 입력하여 설정합니다.
XML : unique_key, Script : UniqueKey  자동저장되는 localstorage key에 해당 값을 추가합니다.
XML : use_manually_save, Script : UseManuallySave  수동 임시 저장 기능의 사용여부를 설정합니다. (0:사용안함, 1:사용함)
XML : use_encrypt, Script : UseEncrypt  localstorage에 자동 저장되는 html의 암호화 여부를 설정합니다. (0:암호화안함, 1:암호화함)
XML : use_manually_save_shortcut_key, Script : UseManuallySaveShortcutKey  에디터에서 ctrl + s 단축키 입력시 수동 임시 저장을 수행하도록 설정합니다.
                                         (단축키의 기본값은 에디터body를 html로 다운로드하는 기능을 수행합니다.)
XML : save_and_start_interval, Script : SaveAndStartInterval  에디터 환경설정에서 자동저장 기능 활성화 시 최초 저장 여부를 설정합니다.(0:즉시 최초저장, 1:interval후 최초 저장)
XML : popup_width, Script : PopupWidth  자동 저장 내용 불러오기 팝업 창의 너비를 설정합니다.
XML : popup_height, Script : PopupHeight  자동 저장 내용 불러오기 팝업 창의 높이를 설정합니다.

sample code

에디터에서 자동저장에 관련된 설정을 적용합니다.

XML

.....
<setting>
    .....
    <set_auto_save interval="10"
        max_count="3"
        unique_key="autosaved"
        use_encrypt="1"
        use_manually_save="1"
        use_manually_save_shortcut_key="1"
        save_and_start_interval="0"
        popup_width="500"
        popup_height="400">1</set_auto_save>
    .....
</setting>
.....
