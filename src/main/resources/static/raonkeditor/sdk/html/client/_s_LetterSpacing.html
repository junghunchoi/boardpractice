K Editor :: LetterSpacing

XML : <letter_spacing>.....</letter_spacing>, Script : LetterSpacingList

Script 설정이 우선 적용 됩니다.
에디터 글자간격 메뉴에 노출할 크기 리스트를 설정합니다.

remarks

XML:letter_spacing  필요 시 <spacing>.....</spacing> 항목을 추가하여 설정합니다.
Config:LetterSpacingList  필요 시 LetterSpacingList = "Spacing값";에 콤마(,)로 구분하여 설정합니다.

sample code

에디터 글자간격 메뉴에 노출할 크기를 0.5pt, 1pt, 1.5pt 로 설정합니다.

XML

.....
<letter_spacing inc_dec_gap="1">
    .....
    <spacing>0.5</spacing>
    <spacing>1</spacing>
    <spacing>1.5</spacing>
    .....
</letter_spacing>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            LetterSpacingList: '0.5pt,1pt,1.5pt'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
