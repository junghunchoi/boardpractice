K Editor :: UsePasteToolbarAndContext

XML : <use_paste_toolbar_and_context>.....</use_paste_toolbar_and_context>
Script : UsePasteToolbarAndContext

Script 설정이 우선 적용 됩니다.
에디터 아이콘 또는 마우스 우클릭으로 붙여넣기 사용 여부를 설정합니다.

remarks

기본값은 "0" 이고,
"1" 인 경우 에디터 아이콘 또는 마우스 우클릭으로 붙여넣기 사용함을 의미합니다.

<제약사항>
1. https 사이트에서만 적용됩니다.
(http 사이트에서는 권한 획득이 되지 않아 툴바 또는 우클릭으로 붙여넣기 할 수가 없습니다.)

2. 문서(엑셀, 워드, 한글 등) 복사 붙여넣기 시 원본과 유사하게 보정하는 작업을 할 수 없습니다.
툴바 또는 우클릭으로 붙여넣기 한 경우 클립보드의 저장된 html 내용을 전부 가져올 수가 없기 때문에
원본과 유사하게 붙여넣기 되도록 보정작업을 할 수가 없습니다.
따라서 문서 내용을 복사 후 Ctrl + v로 붙여넣기 할 때보다 원본 문서 호환성이 떨어지게 됩니다

sample code

에디터 아이콘 또는 마우스 우클릭으로 붙여넣기를 사용합니다.

XML

.....
<setting>
    .....
    <use_paste_toolbar_and_context>1</use_paste_toolbar_and_context>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

        var raonkParam = {
            .....

        }

        new RAONKEditor(raonkParam);
    </script>
