K Editor :: RAONKEDITOR_MouseEvent

void RAONKEDITOR_MouseEvent(editorID, paramObj)

에디터 디자인 영역에서 마우스 동작 시 이벤트가 발생합니다.

return value

없음.

parameters

editorID  에디터의 ID를 의미합니다.
paramObj.strEventName  이벤트 이름을 의미합니다.
paramObj.objEvent  이벤트 객체를 의미합니다.
paramObj.objElement  이벤트가 발생한 요소를 의미합니다.

remarks

에디터의 설정 파일(raonkeditor/config/raonk_editor.xml)의 return_event 설정의 mouse_event 속성 값이 1인 경우 이벤트가 발생합니다.
