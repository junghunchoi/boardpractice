K Editor :: CellSelectionMode

XML : <cell_selection_mode>.....</cell_selection_mode>, Script : CellSelectionMode

Script 설정이 우선 적용 됩니다.
셀을 셀렉션 할 때 텍스트를 포함하여 셀렉션 하도록 설정합니다.

remarks

기본값은 "0" 이고 설정 안함,
"1"로 설정 시 셀을 셀렉션 할 때 텍스트를 포함하여 셀렉션 하도록 설정합니다.

sample code

셀을 셀렉션 할 때 텍스트를 포함하여 셀렉션 하도록 설정합니다.
XML

.....
<setting>
    .....
    <cell_selection_mode>1</cell_selection_mode>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
