K Editor :: ZoomList

XML : <zoom>.....</zoom>, Script : ZoomList

Script 설정이 우선 적용 됩니다.
우측 하단의 확대/축소 설정 콤보박스에 표시할 비율을 설정합니다.

remarks

Script 설정 시 두개 이상인 경우 ,(쉼표)로 구분합니다.

sample code

script 설정시 확대/축소 비율을 콤마(,)로 구분하여 입력합니다.

XML

.....
<zoom>
    .....
    <item>50%</item>
    <item>75%</item>
    <item>100%</item>
    <item>125%</item>
    <item>150%</item>
    <item>175%</item>
    <item>200%</item>
    .....
</zoom>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ZoomList: '50%,75%,100%,125%,150%,175%,200%'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
