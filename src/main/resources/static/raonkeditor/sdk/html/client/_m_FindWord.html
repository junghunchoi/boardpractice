K Editor :: FindWord

void FindWord(options, editorName)

에디터에 작성한 내용에서 단어를 찾습니다.

return value

없음.

parameters

options.searchText  찾을 단어를 의미합니다.
options.caseSensitive  대소문자 구분 여부를 의미합니다.
options.revertEndDocument  문서 끝에서 되돌리기 여부를 의미합니다.
options.wordByword  단어 단위로 찾기 여부를 의미합니다.
options.searchDirection  찾을 방향을 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

options.caseSensitive가 true인 경우 대소문자 구분, false인 경우 대소문자 구분하지 않음을 의미합니다.
options.revertEndDocument가 true인 경우 문서 끝에서 되돌림, false인 경우 문서 끝에서 되돌리지 않음을 의미합니다.
options.wordByword가 true인 경우 단어 단위로 찾음 (조사 미포함), false인 경우 단어 단위로 찾지 않음을 의미합니다. (조사 포함)
options.searchDirection가 true인 경우 위에서 아래, false인 경우 아래에서 위를 의미합니다.

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function findWord() {
        var Obj = {
            searchText:'RAONK', 
            caseSensitive:false, 
            revertEndDocument:true,
            wordByWord:false, 
            searchDirection:true, 
            initFocus:false
        }

        // "RAONK"라는 문자열을 id가 editor1인 에디터에서 위에서 아래방향으로 문서 끝이면 되돌리고
        // 찾은 위치로 커서를 변경하면서 대소문자 구분없이 문자단위로 찾습니다.
        RAONKEDITOR.findWord(Obj, 'editor1');
    }
</script>
