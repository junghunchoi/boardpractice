K Editor :: Figure

XML : <figure use="">.....</figure>, Script : Figure.Use
XML : <figure figure_style="">.....</figure>, Script : Figure.FigureStyle
XML : <figure figcaption_style="">.....</figure>, Script : Figure.FigcaptionStyle
XML : <figure default_caption="">.....</figure>, Script : Figure.DefaultCaption

Script 설정이 우선 적용 됩니다.
이미지 추가시 이미지 설명에 대한 설정들을 적용합니다.

remarks

XML : use, Script : Use  기본값은 "0" 이고 사용 안함, "1"으로 설정 시 이미지 추가할 때 이미지의 설명넣기 기능을 추가합니다.
XML : figure_style, Script : FigureStyle  css style을 입력하여 figure tag의 style을 설정합니다.
XML : figcaption_style, Script : FigcaptionStyle  css style을 입력하여 figcaption의 style을 설정합니다.
XML : default_caption, Script : DefaultCaption  기본값은 "" 이고 설정 안함, 텍스트값 입력시 이미지 설명 추가시 기본값을 설정합니다.

sample code

이미지 추가시 이미지 설명에 대한 설정들을 적용합니다.

XML

.....
<setting>
    .....
    <figure use="1" figure_style="border:solid gray" figcaption_style="color:blue;font-size:14pt" default_caption="K Editor"></figure>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
