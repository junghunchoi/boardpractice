K Editor :: SetFontColor

void SetFontColor(color, editorName)

에디터 디자인 영역에서 선택 영역의 글자 색을 설정합니다.

return value

없음.

parameters

color  설정할 글자 색을 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

color에는 색 이름, HEX 값, rgb()를 포함한 RGB 값을 입력할 수 있습니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function setFontColor() {
        // 선택한 영역의 글자 색을 '파란색'으로 설정합니다.
        var color = 'blue';

        RAONKEDITOR.setFontColor(color, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
