K Editor :: EditingAreaBgColor

XML : <editng_area_bg_color>.....</editng_area_bg_color>, Script : EditingAreaBgColor

Script 설정이 우선 적용 됩니다.
에디터 작성영역(Iframe)의 배경색을 설정합니다.

remarks

CSS background-color 속성값으로 설정합니다.

sample code

에디터 작성영역(Iframe)의 배경색을 #4284c6 으로 설정합니다.

XML

.....
<setting>
    .....
    <editng_area_bg_color>#4284c6</editng_area_bg_color>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            EditingAreaBgColor: '#4284c6' // RGB 값으로도 설정 가능합니다. [ex : 'RGB(66,132,198)']
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
