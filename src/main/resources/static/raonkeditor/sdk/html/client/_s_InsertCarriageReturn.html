K Editor :: InsertCarriageReturn

XML : <insert_carriage_return>.....</insert_carriage_return>, Script : InsertCarriageReturn

Script 설정이 우선 적용 됩니다.
get Api 호출 시 p태그와 br태그뒤에 carriage_return 삽입여부를 설정합니다.

remarks

RFC2822 규약으로 인한 줄당 글자수 제한에 대해 해당 설정이 사용됩니다.
기본값은 "0" 이며 사용 안함,
"1" 로 설정 시 get Api 호출 시 p태그와 br태그뒤에 carriage_return을 삽입합니다.

sample code

get Api 호출 시 p태그와 br태그뒤에 carriage_return을 삽입합니다.

XML

.....
<setting>
    .....
    <insert_carriage_return>1</insert_carriage_return>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">

            .....

        }

    </script>
</div>
