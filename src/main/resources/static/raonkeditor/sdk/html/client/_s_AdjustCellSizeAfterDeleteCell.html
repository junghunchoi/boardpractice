K Editor :: AdjustCellSizeAfterDeleteCell

XML : <adjust_cell_size_after_delete_cell>.....</adjust_cell_size_after_delete_cell>,
Script : AdjustCellSizeAfterDeleteCell

Script 설정이 우선 적용 됩니다.
표 가로줄, 세로줄 삭제 후 td높이에 따라 셀들의 크기를 조정해 주도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 표 가로줄, 세로줄 삭제 후 td높이에 따라 셀들의 크기를 셀 내용에 맞춰 조정 합니다.

sample code

표 가로줄, 세로줄 삭제 후 td높이에 따라 셀들의 크기를 셀 내용에 맞춰 조정 합니다.

XML

.....
<setting>
    .....
    <adjust_cell_size_after_delete_cell>1</adjust_cell_size_after_delete_cell>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->
