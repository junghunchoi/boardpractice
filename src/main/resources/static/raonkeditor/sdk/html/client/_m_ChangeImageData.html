K Editor :: ChangeImageData

void ChangeImageData(valueObject, editorName)

선택한 이미지를 다른 이미지로 변경합니다.

return value

없음.

parameters

valueObject.src  이미지 URL을 의미합니다.
valueObject.width  이미지 너비를 의미합니다.
valueObject.height  이미지 높이를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function changeImageData() {
        var url = 'http://www.raonk.com/image/newImage.jpg';

        RAONKEDITOR.changeImageData({src : url, width : '900', height : '550'}, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
