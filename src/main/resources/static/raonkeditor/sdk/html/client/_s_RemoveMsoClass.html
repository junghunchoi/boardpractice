K Editor :: RemoveMsoClass

XML : <remove_mso_class>.....</remove_mso_class>, Script : RemoveMsoClass

Script 설정이 우선 적용 됩니다.
set API사용 또는 붙여넣기 시 Mso로 시작하는 클래스를 지울지에 대하여 설정합니다.
마지막에 연속으로 br태그가 있는 경우는 제외합니다.

remarks

기본값은 "1"로 set 또는 붙여넣기 시 Mso로 시작하는 클래스를 제거함,
"0"으로 설정 시 제거하지 않습니다.
제거하고싶지 않은 Mso로 시작하는 태그가 존재할시 활용 가능합니다.

sample code

set API사용 또는 붙여넣기 시 Mso로 시작하는 클래스를 제거하지 않습니다.

.....
<setting>
    .....
    <remove_mso_class>0</remove_mso_class>
    .....
</setting>
