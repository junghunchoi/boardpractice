K Editor :: PasteImageBase64

XML : <paste_image_base64>.....</paste_image_base64>, Script : PasteImageBase64
XML : <paste_image_base64 remove="">.....</paste_image_base64>, Script : PasteImageBase64Remove
XML : <paste_image_base64 view_base64_source="">.....</paste_image_base64>, Script : ViewImgBase64Source

Script 설정이 우선 적용 됩니다.
XML : paste_image_base64, Script : PasteImageBase64  이미지 붙여넣기 시 이미지를 base64 형태로 변환 여부를 설정합니다.
XML : remove, Script : PasteImageBase64Remove  이미지 붙여넣기 시 base64 형태 이미지 삭제 여부를 설정합니다.
XML : view_base64_source, Script : ViewImgBase64Source  HTML 모드에서 이미지 src를 base64 형태로 보기 여부를 설정합니다.

remarks

XML : paste_image_base64, Script : PasteImageBase64  0인 경우 변환하지 않음, 1인 경우 변환함을 의미합니다.
XML : remove, Script : PasteImageBase64Remove  0인 경우 삭제하지 않음, 1인 경우 삭제함을 의미합니다.
XML : view_base64_source, Script : ViewImgBase64Source  0인 경우 보지 않음, 1인 경우 보기를 의미합니다.
