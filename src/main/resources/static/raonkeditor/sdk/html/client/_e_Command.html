K Editor :: RAONKEDITOR_CommandEvent

void RAONKEDITOR_CommandEvent(editorID, paramObject)

에디터 메뉴 클릭 및 실행 시 이벤트가 발생합니다.

return value

없음.

parameters

editorID  에디터의 ID를 의미합니다.
paramObject  Command Name과 실행 및 종료 정보를 담은 객체 입니다.

remarks

에디터의 설정 파일(raonkeditor/config/raonk_editor.xml)의 return_event 설정의 command_event 속성 값이 1인 경우 이벤트가 발생합니다.
paramObject는 아래의 정보를 담고 있습니다.
 - paramObject.cmd = 이벤트가 발생한 에디터 기능을 의미합니다.
 - paramObject.tool = true : 기능시작 혹은 실행전 / false : 동작완료 혹은 취소
