K Editor :: AllowInsertFileType

XML : <allow_insert_file_type>.....</allow_insert_file_type>, Script : AllowInsertFileType
XML : <allow_insert_file_type max_file_size="....."></allow_insert_file_type>, Script : MaxInsertFileSize

Script 설정이 우선 적용됩니다.
파일 삽입 시 허용할 확장자를 설정합니다.
파일 삽입 시 허용할 최대 파일 사이즈를 제한합니다.

remarks

XML : allow_insert_file_type, Script : AllowInsertFileType   확장자를 ","(콤마)로 구분하여 입력합니다.
XML : max_file_size, Script : MaxInsertFileSize   용량을 바이트단위까지 입력합니다.

sample code

파일 삽입 시 허용할 확장자를 "docx,pptx"로 설정합니다.
파일 삽입 시 허용할 최대 파일 사이즈를 "10MB"로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <allow_insert_file_type>docx,pptx</allow_insert_file_type>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->
