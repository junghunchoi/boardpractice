K Editor :: SetReadOnly

void SetReadOnly(isReadOnly, actvationList, editorName)

에디터를 읽기 전용으로 변경 여부를 설정합니다.

return value

없음.

parameters

isReadOnly  읽기 전용 여부를 의미합니다.
actvationList  활성화할 에디터 메뉴를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

isReadOnly가 true인 경우 읽기 전용으로 변경, false인 경우 읽기 전용으로 변경하지 않음을 의미합니다.
actvationList가 두개 이상인 경우 ,(쉼표)로 구분합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function setReadOnly() {
        // 에디터를 읽기 전용으로 변경 및 '새 문서', '파일로 저장' 메뉴를 활성화합니다.
        var actvationList = 'new,save';

        RAONKEDITOR.setReadOnly(true, actvationList, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
