K Editor :: IeBugFixed

XML : <ie_remove_hyfont>.....</ie_remove_hyfont>, Script : IeBugFixedHyfont
XML : <ie_remove_hyfont replace_font="">.....</ie_remove_hyfont>, Script : IeBugFixedHyfontReplaceFont
XML : <apply_all_browser>.....</apply_all_browser>, Script : IeBugFixedApplyAllBrowser

Script 설정이 우선 적용 됩니다.
XML : ie_remove_hyfont, Script : IeBugFixedHyfont  IE11브라우저에서 에디터 사용 시 HY폰트의 깨짐 현상을 보정 여부를 설정합니다.
XML : replace_font, Script : IeBugFixedHyfontReplaceFont  IE11브라우저에서 에디터 사용 시 HY폰트를 삭제하거나 대체 여부를 설정합니다.
XML : delete_table_align, Script : IeBugFixedApplyAllBrowser  테이블 align 제거와 관련하여 IE브라우저 및 타 브라우저가 동일하게 반영 여부를 설정합니다.

remarks

XML : ie_remove_hyfont, Script : IeBugFixedHyfont  기본값은 "0" 이고 보정 안함, "1"로 설정 시 보정합니다.
XML : replace_font, Script : IeBugFixedHyfontReplaceFont  기본값은 "" 이고 삭제, 글꼴명 설정 시 해당 글꼴로 교체합니다.
XML : apply_all_browser, Script : IeBugFixedApplyAllBrowser  기본값은 "0" 이고 반영 안함, "1"로 설정 시 반영합니다.

sample code

IE11브라우저에서 에디터 사용 시 HY폰트의 깨짐 현상을 보정합니다.
테이블 align 제거와 관련하여 IE브라우저 및 타 브라우저가 동일하게 반영되도록 설정합니다.
