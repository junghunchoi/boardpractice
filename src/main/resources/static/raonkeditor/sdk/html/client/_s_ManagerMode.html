[관리자 라이선스 전용]

K Editor :: ManagerMode

XML : <manager_mode use=""></manager_mode>, Script : ManagerMode

Script 설정이 우선 적용 됩니다.
관리자기능(Form 요소 관리 및 Table Locker)활성화 여부를 설정합니다.

remarks

0인 경우 관리자 모드 활성화하지 않음, 1인 경우 관리자 모드 활성화함을 의미합니다.

sample code

관리자 모드를 사용합니다.

XML

.....
<setting>
    .....
    <manager_mode use="1"></manager_mode>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ManagerMode: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
