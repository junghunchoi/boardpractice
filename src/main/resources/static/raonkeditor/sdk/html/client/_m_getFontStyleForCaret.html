K Editor :: GetFontStyleForCaret

object GetFontStyleForCaret(editorName)

에디터 디자인 영역에서 커서 위치에 있는 폰트 스타일(fontFamily, fontSize, lineHeight)을 반환해줍니다.

return value

에디터 디자인 영역에서 커서 위치에 있는 글꼴(fontFamily), 글자 크기(fontSize), 줄 간격(lineHeight)

parameters

editorName  API를 적용할 에디터의 id를 의미합니다.

remarks

없음.

        function getFontStylesForCaret() {
            // id가 editor1인 에디터의 에디터 디자인 영역에서 커서 위치에 있는 폰트 스타일(fontFamily, fontSize, lineHeight)을 반환받습니다.
            var fontStyleObject = RAONKEDITOR.getFontStyleForCaret('editor1');

            console.log("lineHeight = " + fontStyleObject.lineHeight); // 커서 위치에 있는 줄 간격(lineHeight)
    }
</script>
