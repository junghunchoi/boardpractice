K Editor :: TableTemplateListUrl

XML : <template_list_url>.....</template_list_url>, Script : TableTemplateListUrl
XML : <template_list_url use_basic_template="">.....</template_list_url>, Script : UseBasicTemplate

Script 설정이 우선 적용 됩니다.
XML : template_list_url, Script : TableTemplateListUrl  웹으로 접근 가능한 리턴 XML 리턴 타입의 사용자 표 템플릿 url을 경로를 설정합니다.
XML : use_basic_template, Script : UseBasicTemplate  기본 표 템플릿 사용 여부를 설정합니다.

remarks

XML : use_basic_template, Script : UseBasicTemplate  0인 경우 기본 표 템플릿 사용 안 함, 1인 경우 기본 표 템플릿 사용함을 의미합니다.

sample code

사용자 표 템플릿 경로를 설정하고, 기본 표 템플릿을 사용하지 않습니다.
