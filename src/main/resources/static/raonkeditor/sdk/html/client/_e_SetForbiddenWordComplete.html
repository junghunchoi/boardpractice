K Editor :: RAONKEDITOR_SetForbiddenWordComplete

void RAONKEDITOR_SetForbiddenWordComplete(editorID, evtDataObj)

금지어 추출 팝업이 적용 또는 닫기로 닫힌 후 이벤트가 발생합니다.

return value

없음.

parameters

editorID  에디터의 ID를 의미합니다.
evtDataObj.permitWord  삭제하지 않은 금지어를 의미합니다. (중복 허용)
evtDataObj.permitWordToDeduplication  삭제하지 않은 금지어를 의미합니다. (중복 비허용)
evtDataObj.callbackType  API인 경우 API로 금지어 추출, ICON인 경우 아이콘으로 금지어 추출을 의미합니다.
evtDataObj.callbackStatus  Apply인 경우 금지어 추출 적용, Cancel인 경우 금지어 추출 팝업 창 닫기를 의미합니다.

remarks

에디터의 설정 파일(raonkeditor/config/raonk_editor.xml)의 forbidden_word 설정의 mode 속성 값이 1인 경우 이벤트가 발생합니다.
