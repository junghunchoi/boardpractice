remarks

XML : word_count, Script : Use  기본값은 "0"으로 미표시, "1"로 설정시 글자수와 행을 우측 하단에 표시합니다.
XML : limit, Script : Limit  글자수의 제한여부를 설정합니다. ("0":제한안함, "1":제한함)
XML : limit_char, Script : LimitChar  제한할 글자의 Byte를 설정(숫자입력)합니다. (limit_char의 값이 설정되어있으면 limit_char의 설정을 우선으로 합니다.)
XML : limit_byte, Script : LimitByte  제한할 글자수를 설정(숫자입력)합니다.
XML : limit_line, Script : LimitLine  입력 제한할 행의 수를 설정(숫자입력)합니다.
XML : count_white_space, Script : CountWhiteSpace  문자수에 공백을 포함할지 설정합니다. ("0":포함안함, "1":포함)
XML : limit_message, Script : LimitMessage  글자수 또는 행의 제한 초과시 alert의 출력 여부를 설정합니다. ("0":출력안함, "1":출력 / WordCount.Limit="1" 일때 작동합니다.)
