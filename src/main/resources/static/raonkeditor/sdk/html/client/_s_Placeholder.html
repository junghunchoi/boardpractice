K Editor :: Placeholder

XML : <placeholder content="">.....</placeholder>, Script : Placeholder.Content
XML : <placeholder font_color="">.....</placeholder>, Script : Placeholder.FontColor
XML : <placeholder font_size="">.....</placeholder>, Script : Placeholder.FontSize
XML : <placeholder font_family="">.....</placeholder>, Script : Placeholder.FontFamily

Script 설정이 우선 적용 됩니다.
에디터 Body 내부에 아무글도 작성 안했을 때 표시되는 기본문구의 내용 및 CSS를 설정합니다.

remarks

XML : content, Script : Content  출력할 문구를 설정합니다.
XML : font_color, Script : FontColor  Placeholder의 폰트 색상을 설정합니다.
XML : font_size, Script : FontSize  Placeholder의 폰트 크기를 설정합니다.
XML : font_family, Script : FontFamily  Placeholder의 글꼴을 설정합니다.

sample code

에디터 Body 내부에 아무글도 작성 안했을 때 표시되는 기본문구의 내용 및 CSS를 설정합니다.

XML

.....
<setting>
    .....
    <placeholder content="내용을 입력하세요." font_color="rgb(50,100,140)" font_size="20pt" font_family="Arial">placeholder</placeholder>
    .....
<setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            Placeholder: {
                Content: '내용을 입력하세요.',
                FontColor: 'rgb(50,100,140)',
                FontSize: '20pt',
                FontFamily: 'Arial'
            }
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
