K Editor :: RAONKEDITOR_BeforeInsertHyperlink

string RAONKEDITOR_BeforeInsertHyperlink(editor, linkUrl)

하이퍼링크 삽입 전 이벤트가 발생합니다.

return value

파라미터로 받은 linkUrl을 리턴 합니다.

parameters

editor  에디터의 id입니다.
linkUrl  삽입할 URL을 의미합니다.

remarks

파라미터로 받은 linkUrl을 다른 URL로 변경한 후 리턴 하면 변경한 URL이 삽입됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">

    function RAONKEDITOR_BeforeInsertHyperlink(editor, linkUrl) {
        // 에디터에 하이퍼링크를 삽입하기 전 해당 이벤트가 호출됩니다. 
        // 이 부분에서 url을 가공 처리 하신 후 리턴 합니다.

        return linkUrl;
    }

</script>
