K Editor :: SetHeightForDisplay

void SetHeightForDisplay(editorName)

view 모드일 때 에디터의 내용에 따라 에디터의 크기가 늘어납니다.

return value

없음.

parameters

editorName  에디터의 ID를 의미합니다.

remarks

에디터의 설정 파일(raonkeditor/config/raonk_editor.xml)의
view_mode_auto_width 또는 view_mode_auto_height 설정 값을 1로 설정해야 됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function setHeightForDisplay() {
        RAONKEDITOR.setHeightForDisplay('editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
