K Editor :: RAONKEDITOR_AgentInstall

boolean RAONKEDITOR_AgentInstall(editorID, paramObj)

agent 설치 파일 다운로드 시 이벤트가 발생합니다.

return value

true인 경우 설치 파일 다운로드, false인 경우 설치 파일 다운로드 하지 않음을 의미합니다.

parameters

editorID  에디터의 ID를 의미합니다.
paramObj.type  install인 경우 설치, update인 경우 업데이트를 의미합니다.

remarks

없음.
