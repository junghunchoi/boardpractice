K Editor :: SetLockCommand

void SetLockCommand(cmd, lock, editorName)

에디터 메뉴의 잠금 여부를 설정합니다. 잠금 설정 시 메뉴 실행을 막습니다.

return value

없음.

parameters

cmd  에디터 메뉴를 의미합니다.
lock  잠금 여부를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

lock이 true일 경우 메뉴 잠금, false일 경우 메뉴 잠금 해제를 의미합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function setLockCommand() {
        // '새 문서' 메뉴의 실행을 막습니다.
        var cmd = 'new';

        RAONKEDITOR.setLockCommand(cmd, true, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
