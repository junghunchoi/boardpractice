K Editor :: InputSelect

void InputSelect(editorName)

펼침 목록 팝업 창이 열립니다.

return value

없음.

parameters

editorName  에디터의 ID를 의미합니다.

remarks

펼침 목록 팝업 창에서 select 태그를 생성할 수 있습니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function inputSelect() {
        RAONKEDITOR.inputSelect('editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
