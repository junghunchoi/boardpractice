[IE 전용]

K Editor :: AllowUnableToDeleteMsg

XML : <allow_unable_to_delete_msg>.....</allow_unable_to_delete_msg>, Script : AllowUnableToDeleteMsg

Script 설정이 우선 적용 됩니다.
IE 브라우저에서 선택 영역 삭제시에 표의 일부가 들어있을때 알림메세지를 출력하도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 IE 브라우저에서 선택 영역 삭제시에 표의 일부가 들어있을때 알림메세지를 출력 합니다.

sample code

IE 브라우저에서 선택 영역 삭제시에 표의 일부가 들어있을때 알림메세지를 출력합니다.

XML

.....
<setting>
    .....
    <allow_unable_to_delete_msg>1</allow_unable_to_delete_msg>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
