K Editor :: PluginToolEx

XML : <plugin_ex>.....</plugin_ex>, Script : PluginToolEx

Script 설정이 우선 적용 됩니다.
플러그인 도구 목록을 설정합니다.

remarks

autogrow인 경우 내용 길이에 따라 에디터 길이 자동 증가,
intelligent인 경우 테이블 위에 <p> 태그가 없을 시 클릭 하면 <p> 생성을 의미합니다.
Script 설정 시 두개 이상인 경우 ,(쉼표)로 구분합니다.

mention인 경우 멘션 기능을 의미합니다. (IE 제외)

sample code

내용 길이에 따라 에디터 길이 자동 증가, 테이블 위에 p 태그가 없을 시 클릭 하면 p 생성 기능을 사용합니다.
