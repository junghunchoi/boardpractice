K Editor :: UploadMethod

XML : <uploader_setting method="">.....</uploader_setting>, Script : UploadMethod

Script 설정이 우선 적용 됩니다.
이미지 업로드시 서버에 업로드 할지, 서버에 업로드 하지않고 이미지 태그 src에 base64로 추가할지 설정합니다.

remarks

기본값은 "upload" 이며 이미지 업로드시 서버에 이미지를 업로드 하고 이미지 태그의 src를 해당 경로로 지정,
"base64" 로 설정하는 경우, 이미지 업로드시 서버에 이미지를 업로드 하지 않고 이미지 태그의 src를 해당 이미지의 base64로 지정합니다.

sample code

이미지 업로드시 서버에 이미지를 업로드 하지 않고 이미지 태그의 src를 해당 이미지의 base64로 지정합니다.

XML

.....
<uploader_setting>
    .....
    <uploader_setting method="base64"></uploader_setting>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UploadMethod: 'base64'
            }
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
