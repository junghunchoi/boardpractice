K Editor :: WebFontCssUrl

XML : <web_font_css_url>.....</web_font_css_url>, Script : WebFontCssUrl
XML : <web_font_css_url always_set="">.....</web_font_css_url>, Script : WebFontCssAlwaysSet

Script 설정이 우선 적용 됩니다.
에디터 로드시 사용자의 web font CSS를 에디터 body 및 글꼴 메뉴의 head 부분에 import 해주도록 설정합니다.
에디터 작성영역 및 글꼴메뉴(콤보박스)에 적용됩니다.

remarks

XML : web_font_css_url, Script : WebFontCssUrl  import할 사용자의 CSS url(http://를 포함한 전체)을 입력합니다.
XML : always_set, Script : WebFontCssAlwaysSet  해당 CSS를 항상 포함할지의 여부를 설정합니다. (0=최초로드시에만 포함, 1=항상포함)

sample code

에디터 로드시 사용자의 web font CSS를 에디터 body의 head부분에 import후 항상 포함해주도록 설정합니다.

.....
<setting>
    .....
    <web_font_css_url always_set="1">.....</web_font_css_url>
    .....
