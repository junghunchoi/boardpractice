K Editor :: UseHtmlProcessByWorker

XML : <use_htmlprocess_worker>.....</use_htmlprocess_worker>, Script : UseHtmlProcessByWorker
XML : <use_htmlprocess_worker use_set_api="">.....</use_htmlprocess_worker>, Script : UseHtmlProcessByWorkerSetApi

Script 설정이 우선 적용 됩니다.
HTML 처리 및 set API 사용시 Web Worker의 사용 여부를 설정합니다.

remarks

XML : use_htmlprocess_worker, Script : UseHtmlProcessByWorker  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 HTML 처리시 Web Worker를 사용합니다.
XML : use_set_api, Script : UseHtmlProcessByWorkerSetApi  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 set API 사용시 Web Worker를 사용합니다.

sample code

HTML 처리 및 set API 사용시 Web Worker를 사용하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_htmlprocess_worker use_set_api="1">1</use_htmlprocess_worker>
    .....
</setting>
.....

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
