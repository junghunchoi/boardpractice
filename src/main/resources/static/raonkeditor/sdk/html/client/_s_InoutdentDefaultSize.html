K Editor :: InoutdentDefaultSize

XML : <inoutdent default_size="">.....</inoutdent>, Script : InoutdentDefaultSize

Script 설정이 우선 적용 됩니다.
들여쓰기 및 내어쓰기의 기본 크기값을 설정합니다.

remarks

들여쓰기 및 내어쓰기의 기본 크기값을 설정합니다.
값 + "px"을 입력하여 설정합니다.

sample code

들여쓰기 및 내어쓰기의 기본 크기값을 50px로 설정합니다.

XML

.....
<setting>
    .....
    <inoutdent default_size="50px"></inoutdent>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            InoutdentDefaultSize: '50px'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
