K Editor :: MiniPhotoEditor

XML : <width>.....</width>, Script : MiniPhotoEditor.Width
XML : <height>.....</height>, Script : MiniPhotoEditor.Height
XML : <background_color>.....</background_color>, Script : MiniPhotoEditor.BackgroundColor
XML : <background_opacity>.....</background_opacity>, Script : MiniPhotoEditor.BackgroundOpacity
XML : <img_tag_remove_attribute>.....</img_tag_remove_attribute>, Script : MiniPhotoEditor.ImgTagRemoveAttribute

Script 설정이 우선 적용 됩니다.
미니 이미지 편집기의 각종 설정을 적용합니다.

remarks

XML : width, Script : Width  미니 이미지 편집기 팝업의 가로 길이를 설정합니다.
XML : height, Script : Height  미니 이미지 편집기 팝업의 세로 길이를 설정합니다.
XML : background_color, Script : BackgroundColor  미니 이미지 편집기 팝업의 뒷 배경 색상을 설정합니다.
XML : background_opacity, Script : BackgroundOpacity  미니 이미지 편집기 팝업의 뒷 배경 투명도를 설정합니다. (0.0~1.0)
XML : img_tag_remove_attribute, Script : ImgTagRemoveAttribute  미니 이미지 편집기로 편집 시 특정 프로퍼티를 removeAttribute 하도록 설정합니다.                                                                                        (콤마(,)로 구분하여 attribute명 입력)

sample code

미니 이미지 편집기의 각종 설정을 적용합니다.

XML

.....
<mini_photo_editor>
    .....
    <width>800px</width>
    <height>600px</height>
    <background_color>rgb(50,100,140)</background_color>
    <background_opacity>0.7</background_opacity>
    <img_tag_remove_attribute>alt,class</img_tag_remove_attribute>
    .....
<mini_photo_editor>
.....
