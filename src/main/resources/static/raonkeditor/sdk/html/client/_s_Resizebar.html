K Editor :: Resizebar

XML : <resizebar>.....</resizebar>, Script : ResizeBar

XML : <resizebar holder_sync=".....">.....</resizebar>, Script : ResizeBarHolderSync

에디터 높이를 조절 기능 사용여부 및 EditorHolder로 지정한 영역에 height 값을 설정합니다.

remarks

resizebar   기본값은 "0" 이고 높이 조절 미사용, "1"로 설정 시 에디터 높이 조절 사용으로 설정합니다.
resizebar@holder_sync   기본값은 "0" 이고 EditorHolder 영역에 height를 설정하지 않고,"1"로 설정 시 EditorHolder 영역에도 height를 설정합니다.
table 구조에 에디터 EditorHolder를 지정할 경우 리사이즈 시 하단 영역이 가려지는 상황일 때 height를 설정하여 사용하면 유용합니다.

sample code

에디터 높이를 조절 기능 사용여부를 사용함 및 Table에 EditorHolder를 지정하여 에디터를 생성 후 리사이즈 시 높이를 설정 할 수 있도록 설정합니다.

XML

.....
<setting>
    .....
    <resizebar holder_sync="1">1</resizebar>
    .....
</setting>
.....
