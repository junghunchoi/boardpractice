K Editor :: DefaultBodySpace

XML : <use_default_body_space>.....</use_default_body_space>, Script : DefaultBodySpace.Use
XML : <use_default_body_space mode="">.....</use_default_body_space>, Script : DefaultBodySpace.Mode
XML : <use_default_body_space value="">.....</use_default_body_space>, Script : DefaultBodySpace.Value

Script 설정이 우선 적용 됩니다.
에디터 작성영역의 기본 여백 사용을 설정합니다.

remarks

XML : use_default_body_space, Script : Use  기본값은 "1"로 작성영역 기본 여백을 사용, "0"으로 설정시 사용하지 않습니다.
XML : mode, Script : Mode  기본값은 "1"로 여백을 margin으로 적용, "2"로 설정시 margin:0px 및 padding으로 적용됩니다. (value의 값이 적용됩니다.)
XML : value, Script : Value  여백으로 사용될 사이즈값을 설정합니다.
                                 상하좌우 여백 설정을 다르게 원하시면 위,오른쪽,아래,왼쪽 순서에 맞게 콤마(,)로 구분하여 입력합니다.

sample code

에디터 작성영역의 기본 여백을 margin:"0px" padding:"30px 30px 30px 30px"로 적용합니다.

XML

.....
<setting>
    .....
    <use_default_body_space mode="2" value="30px">1</use_default_body_space>
    .....
</setting>
.....
