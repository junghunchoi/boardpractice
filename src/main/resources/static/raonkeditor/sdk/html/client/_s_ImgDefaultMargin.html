K Editor :: ImgDefaultMargin

XML : <img_default_margin margin_left="">.....</img_default_margin, Script : ImgDefaultMarginLeft>
XML : <img_default_margin margin_right="">.....</img_default_margin>, Script : ImgDefaultMarginRight
XML : <img_default_margin margin_top="">.....</img_default_margin>, Script : ImgDefaultMarginTop
XML : <img_default_margin margin_bottom="">.....</img_default_margin>, Script : ImgDefaultMarginBottom

Script 설정이 우선 적용 됩니다.
이미지 다이얼로그의 상, 하, 좌, 우 기본 여백을 설정합니다.
이미지 삽입시가 아닌 다이얼로그 팝업시 기본적으로 입력되어있는 값을 설정합니다.

remarks

XML : margin_left, Script : ImgDefaultMarginLeft  이미지 다이얼로그에 입력될 기본 하여백을 설정합니다.
XML : margin_right, Script : ImgDefaultMarginRight  이미지 다이얼로그에 입력될 기본 좌여백을 설정합니다.
XML : margin_top, Script : ImgDefaultMarginTop  이미지 다이얼로그에 입력될 기본 우여백을 설정합니다.
XML : margin_bottom, Script : ImgDefaultMarginBottom  이미지 다이얼로그에 입력될 기본 상여백을 설정합니다.

sample code

이미지 다이얼로그의 상, 하, 좌, 우 기본 여백을 6으로 설정합니다.

XML
.....
<uploader_setting>
    .....
    <img_default_margin margin_left="6" margin_right="6" margin_top="6" margin_bottom="6"></img_default_margin>
    .....
</uploader_setting>
.....
