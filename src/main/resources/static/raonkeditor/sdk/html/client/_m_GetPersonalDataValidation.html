K Editor :: GetPersonalDataValidation

boolean GetPersonalDataValidation(editorName)

개인 정보 검사 시 에디터에 작성한 내용 중 개인 정보 유무를 가지고 옵니다.

return value

true인 경우 개인 정보가 있음, false인 경우 개인 정보가 없음을 의미합니다.

parameters

editorName  에디터의 ID를 의미합니다.

remarks

개인 정보 검사 시 이메일, 전화번호, 주민 등록 번호 유무를 검사합니다.

sample code

    function getPersonalDataValidation() {
        var result = RAONKEDITOR.getPersonalDataValidation('editor1');
    }
</script>
