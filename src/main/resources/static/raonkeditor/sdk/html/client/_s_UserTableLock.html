XML : <table_lock_user_mode default_show_lock_icon=""></table_lock_user_mode>, Script : UserTableLock.DefaultShowLockIcon
XML : <table_lock_user_mode allow_change_mode=""></table_lock_user_mode>, Script : UserTableLock.AllowChangeMode

Script 설정이 우선 적용 됩니다.
XML : default_show_lock_icon_user_mode, Script : Use   잠금 아이콘을 사용자 모드에서 보여주기 여부를 설정합니다.
XML : lock_name, Script : LockName  테이블 및 셀 잠금 시 잠금 이름을 설정합니다.
XML : default_table_lock_mode, Script : DefaultTableLockMode  잠금 셀로 체크할 잠금 이름을 설정합니다.
XML : default_show_lock_icon, Script : DefaultShowLockIcon  잠금의 기본 모드를 설정합니다.
XML : allow_change_mode, Script : AllowChangeMode  HTML 모드로 변경 가능 여부를 설정합니다.

remarks

XML : default_show_lock_icon_user_mode, Script : Use  0인 경우 잠금 아이콘을 사용자 모드에서 보여 주지 않음, 1인 경우 잠금 아이콘을 사용자 모드에서 보여 줌을 의미합니다.
XML : lock_name, Script : LockName  설정된 속성값이 있는 경우 테이블 또는 셀을 잠급니다.
XML : default_table_lock_mode, Script : DefaultTableLockMode  여러 개인 경우 콤마 구분자로 넣어주시면 됩니다.
XML : default_show_lock_icon, Script : DefaultShowLockIcon  1인 경우 리사이즈 잠금, 2인 경우 리사이즈 잠금 및 편집 불가를 의미합니다.
XML : allow_change_mode, Script : AllowChangeMode  0인 경우 HTML 모드로 변경 불가능함, 0 이외의 숫자인 경우 HTML 모드로 변경 가능함을 의미합니다.
