Script 설정이 우선 적용 됩니다.
XML : ie11_jaso, Script : Ie11BugFixedJASO  ie11브라우저에서 에디터 사용 시 자소분리 현상 및 align 속성 보정 여부를 설정합니다.
XML : replace_br, Script : Ie11BugFixedReplaceBr  get 할 때에 빈 span에 &nbsp;를 삽입 여부를 설정합니다.
XML : delete_table_align, Script : Ie11BugFixedDeleteTableAlign  ie11브라우저에서 에디터 사용중 표 align 속성을 제거 여부를 설정합니다.
XML : replace_align_margin, Script : Ie11BugFixedReplaceAlignMargin  Set API 사용에서 설정에 의해 align 속성이 제거되면 margin 값으로 대체 여부를 설정합니다.
XML : ie11_typing_bug_in_table, Script : Ie11BugFixedTypingBugInTable  IE11 브라우저 사용중 셀 안의 p태그가 비어있을 경우 br 태그를 추가하여 한글 입력이 오동작 하는 현상을 보정 여부를 설정합니다.

remarks

XML : ie11_jaso, Script : Ie11BugFixedJASO  기본값은 "0" 이고 보정 안함, "1"로 설정 시 보정합니다.
XML : replace_br, Script : Ie11BugFixedReplaceBr  기본값은 "0" 이고 보정 안함, "1"로 설정 시 삽입합니다.
XML : delete_table_align, Script : Ie11BugFixedDeleteTableAlign  기본값은 "0" 이고 제거 안함, "1"로 설정 시 제거합니다.
XML : replace_align_margin, Script : Ie11BugFixedReplaceAlignMargin  기본값은 "0" 이고 보정 안함, "1"로 설정 시 대체합니다.
XML : ie11_typing_bug_in_table, Script : Ie11BugFixedTypingBugInTable  기본값은 "0" 이고 반영 안함, "1"로 설정 시 현상을 보정합니다. (Ie11BugFixedJASO가 1일 때 적용됩니다.)

sample code

ie11브라우저에서 에디터 사용 시 자소분리 현상 및 align 속성 보정 여부를 설정합니다.
IE11 브라우저 사용중 셀 안의 p태그가 비어있을 경우 br 태그를 추가하여 한글 입력이 오동작 하는 현상을 보정합니다.
