K Editor :: GetElementById

object GetElementById(id, editorName)

element의 id로 element를 가지고 옵니다.

return value

element가 리턴 됩니다.

parameters

id  element의 id를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function getElementById() {
        var element = RAONKEDITOR.getElementById('elementId', 'editor1');
    }
</script>
