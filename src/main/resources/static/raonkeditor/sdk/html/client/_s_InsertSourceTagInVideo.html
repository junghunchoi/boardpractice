K Editor :: InsertSourceTagInVideo

XML : <insert_sourcetag_in_video>.....</insert_sourcetag_in_video>, Config : config.InsertSourceTagInVideo

Config 설정이 우선 적용됩니다.
비디오 삽입 시 src 프로퍼티를 사용할지, 내부에 <source> 태그를 삽입하는 방식을 사용할지를 설정합니다.

remarks

allow_video_file_type  기본값은 "0" 이며 src 프로퍼티를 사용하여 파일소스의 정보를 추가하고,
"1" 로 설정 시 내부에 <source> 태그를 삽입하는 방식을 사용합니다.
"1"로 설정 후 파일로 삽입 시 AllowVideoFileType 에 설정된 모든 확장자들이 비디오태그내에 여러 파일 형식 지정으로 추가됩니다.

sample code

비디오 삽입 시 내부에 <source> 태그를 삽입하는 방식을 사용하도록 설정합니다.

XML

.....
<setting>
    .....
    <insert_sourcetag_in_video>1</insert_sourcetag_in_video>
    .....
</setting>
.....
