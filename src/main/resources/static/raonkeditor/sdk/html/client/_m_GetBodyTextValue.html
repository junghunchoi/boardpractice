K Editor :: GetBodyTextValue

void GetBodyTextValue(paramObj, editorName)

에디터에 작성한 내용을 텍스트로 가지고 옵니다.

return value

callback 함수로 에디터 ID와 에디터에 작성한 텍스트가 리턴 됩니다.
strEditorId는 에디터 ID를, strData는 에디터에 작성한 텍스트를 의미합니다.

parameters

paramObj.callback  에디터에 작성한 텍스트를 리턴 받을 callback 함수를 의미합니다.
paramObj.async  
editorName  에디터의 ID를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function getBodyTextValue() {
        // 에디터 ID와 에디터에 작성한 내용을
        var fn_callback = function(value) {
            console.log('에디터 ID : ' + value.strEditorId);
            console.log('작성 내용 : ' + value.strData);
        }
        
        RAONKEDITOR.getBodyTextValue({
            callback : fn_callback,
            async : true
        }, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
