Script : Event.AgentInstall

agent 설치 파일 다운로드 시 발생하는 callback 이벤트를 설정합니다.

remarks

paramObj.type이 install인 경우 설치, update인 경우 업데이트를 의미합니다.
return이 true인 경우 설치 파일 다운로드, false인 경우 설치 파일 다운로드 하지 않음을 의미합니다.
