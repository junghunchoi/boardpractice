K Editor :: SetHorizontalLine

void SetHorizontalLine(url, editorName)

문서 배경 이미지를 삽입합니다. 삽입 시 이미지가 반복되어 삽입됩니다.
에디터 작성 영역에 구분 선을 표시하고 싶은 경우 유용하게 사용할 수 있습니다.

return value

없음.

parameters

url  이미지 URL을 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

없음.
