K Editor :: AllowVideoFileType

XML : <allow_video_file_type>.....</allow_video_file_type, Script : AllowVideoFileType>
XML : <allow_video_file_type max_file_size="">.....</allow_video_file_type>, Script : MaxVideoFileSize
XML : <allow_video_file_type use_poster="">.....</allow_video_file_type>, Script : UseVideoPoster

Script 설정이 우선 적용 됩니다.
XML : allow_video_file_type, Script : AllowVideoFileType  비디오 태그 삽입 시 허용할 확장자를 설정합니다.
XML : max_file_size, Script : MaxVideoFileSize  비디오 태그 삽입 시 허용할 최대 파일 사이즈를 제한합니다.
XML : use_poster, Script : UseVideoPoster  비디오 태그 삽입 팝업 창에서 썸네일 사용 여부를 설정합니다.

remarks

XML : allow_video_file_type, Script : AllowVideoFileType  ","(콤마)로 구분합니다.
XML : max_file_size, Script : MaxVideoFileSize  용량을 바이트단위까지 입력합니다.
XML : use_poster, Script : UseVideoPoster  0인 경우 사용 안 함, 1인 경우 사용함을 의미합니다.

sample code

비디오 태그 삽입 시 허용할 확장자를 "mp4,ogv,webm"로 설정합니다.
비디오 태그 삽입 시 허용할 최대 파일 사이즈를 "30MB"로 설정합니다.
비디오 태그 삽입 팝업 창에서 썸네일을 사용합니다.

.....
<uploader_setting>
    .....
    <allow_video_file_type max_file_size="30MB" use_poster="1">mp4,ogv,webm</allow_video_file_type>
    .....
