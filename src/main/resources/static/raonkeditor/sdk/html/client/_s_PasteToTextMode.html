K Editor :: PasteToTextMode

XML : <paste_to_text_mode>.....</paste_to_text_mode>, Script : PasteToTextMode

Script 설정이 우선 적용 됩니다.
텍스트로 붙여넣기 실행 시 p태그로 감싸지 않고 붙여넣기 하도록 설정합니다.

remarks

기본값은 "0" 이며 텍스트 붙여넣기시 p태그로 감쌈,
"1" 로 설정 시 텍스트 붙여넣기시 p태그로 감싸지 않도록 설정합니다.

sample code

텍스트 붙여넣기시 p태그로 감싸지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <paste_to_text_mode>1</paste_to_text_mode>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            PasteToTextMode: '1'
        }

        new RAONKEditor(raonkParam);
    </script>
</div>
