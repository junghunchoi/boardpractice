remarks

XML : use_personal_setting, Script : UsePersonalSetting  use_personal_setting  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 환경 설정 다이얼로그에 개인화 설정 항목을 활성화 합니다.
                           (자동저장, 자동글머리, 이미지로붙여넣기팝업창, 기본 글꼴, 기본 글자크기, 기본 줄간격)
XML : font_family_use_key_in, Script : PersonalSettingUseFontFamilyKeyin  기본값은 "0" 이며 글꼴 항목에 직접 입력 사용안함, "1" 로 설정 시 글꼴 항목에 직접 입력하는 설정을 추가 합니다.
XML : font_size_use_key_in, Script : PersonalSettingUseFontSizeKeyin  기본값은 "0" 이며 글자크기 항목에 직접 입력 사용안함, "1" 로 설정 시 기본 글자크기 항목에 직접 입력하는 설정을 추가 합니다.
XML : line_height_use_key_in, Script : PersonalSettingUseLineHeightKeyin  기본값은 "0" 이며 기본 줄간격 항목에 직접 입력 사용안함, "1" 로 설정 시 기본 줄간격 항목에 직접 입력하는 설정을 추가 합니다.
