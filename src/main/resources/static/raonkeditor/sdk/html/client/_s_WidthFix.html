K Editor :: WidthFix

XML : <value>.....</value>, Script : WidthFix.Value
XML : <background_color>.....</background_color>, Script : WidthFix.BackgroundColor
XML : <default_view>.....</default_view>, Script : WidthFix.DefaultView

Script 설정이 우선 적용 됩니다.
XML : value, Script : WidthFix.Value  너비를 고정할 width 값을 설정합니다.
XML : background_color, Script : WidthFix.BackgroundColor  너비 고정 후 비활성화 영역의 배경 색상을 설정합니다.
XML : default_view, Script : WidthFix.DefaultView  에디터 로드 시 너비고정을 적용할지 설정합니다.

remarks

XML : value, Script : WidthFix.Value  px 기준으로 설정이 됩니다.
XML : default_view, Script : WidthFix.DefaultView  "0"으로 설정 시 에디터 로드 시 너비고정 안함, "1"로 설정 시 에디터 로드 시 너비고정을 합니다.

sample code

너비고정을 680px로 에디터 로드 시 바로 적용이 되도록 설정합니다.

XML

.....
<width_fix>
    <value>680px</value>
    <background_color>#f0f0f0</background_color>
    <default_view>1</default_view>
</width_fix>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->
