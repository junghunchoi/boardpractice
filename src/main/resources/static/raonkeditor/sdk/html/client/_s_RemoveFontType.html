K Editor :: RemoveFontType

XML : <type>.....</type>, Script : RemoveFontType.Type
XML : <font_family>.....</font_family>, Script : RemoveFontType.FontFamily

Script 설정이 우선 적용 됩니다.
엑셀에서 복사/붙여넣기시 삭제할 type/font_family를 설정합니다.
엑셀에서 복사/붙여넣기 시 monospace로 인해 글꼴 표시가 안되는 현상을 방지하기 위해 사용합니다.

remarks

XML : type, Script : Type  엑셀에서 복사/붙여넣기시 삭제할 type을 설정합니다. (콤마(,)로 구분하여 입력합니다.)
XML : font_family, Script : FontFamily  엑셀에서 복사/붙여넣기시 삭제할 fontfamily를 설정합니다. (콤마(,)로 구분하여 입력합니다.)

sample code

엑셀에서 복사/붙여넣기시 삭제할 type/font_family를 설정합니다.

    .....
    <remove_font_type>

    </remove_font_type>
