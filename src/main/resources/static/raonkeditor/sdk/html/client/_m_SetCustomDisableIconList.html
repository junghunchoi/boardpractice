K Editor :: SetCustomDisableIconList

void SetCustomDisableIconList(disableIconList, editorName)

에디터의 메뉴를 비활성화합니다.

return value

없음.

parameters

disableIconList  비활성화할 에디터 메뉴를 의미합니다.
editorName  에디터의 ID를 의미합니다.

remarks

disableIconList가 두개 이상인 경우 ,(쉼표)로 구분합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>
<script type="text/javascript">
    function setCustomDisableIconList() {
        // 에디터의 '새 문서', '파일로 저장' 메뉴를 비활성화합니다.
        var disableIconList = 'new,save';

        RAONKEDITOR.setCustomDisableIconList(disableIconList, 'editor1');
    }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
    <script type="text/javascript">
        new RAONKEditor({Id:'editor1'});
    </script>
</div>
