K Editor :: RemoveCarriageReturnInTag

XML : <remove_carriage_return_in_tag>.....</remove_carriage_return_in_tag>,
Script : RemoveCarriageReturnInTag

Script 설정이 우선 적용 됩니다.
태그 내부(<태그명 .....>)에 줄바꿈 문자가 있으면 치환하도록 설정합니다.

remarks

기본값은 "1" 이며 태그 내부에 줄바꿈 문자가 있으면 치환하며,
"0"으로 설정시 태그 내부의 줄바꿈 문자를 치환하지 않습니다.

sample code

태그 내부에 줄바꿈 문자가 있는 경우 치환하도록 설정합니다.

XML

.....
<setting>
    .....
    <remove_carriage_return_in_tag>1</remove_carriage_return_in_tag>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->
