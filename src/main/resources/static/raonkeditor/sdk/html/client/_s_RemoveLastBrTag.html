K Editor :: RemoveLastBrTag

XML : <remove_last_br_tag>.....</remove_last_br_tag>, Script : RemoveLastBrTag

Script 설정이 우선 적용 됩니다.
set html API 사용 시 p태그 안에서 마지막 node가 br인 경우 해당 br태그를 삭제하도록 설정합니다.
마지막에 연속으로 br태그가 있는 경우는 제외합니다.

remarks

기본값은 "0"으로 사용 안함,
"1"로 설정 시 set html API 사용 시 p태그 안에서 마지막 node가 br인 경우 해당 br태그를 삭제합니다.

sample code

set html API 사용 시 p태그 안에서 마지막 node가 br인 경우 해당 br태그를 삭제합니다.

XML

.....
<setting>
    .....
    <remove_last_br_tag>1</remove_last_br_tag>
    .....
</setting>
.....
