K Editor :: CleanNestedTagOptions

XML : <remove_tag>.....</remove_tag>, Script : CleanNestedTagOptions.RemoveTag
XML : <nested_count>.....</nested_count>, Script : CleanNestedTagOptions.NestedCount

Script 설정이 우선 적용 됩니다.
중복 태그 제거 사용 시 태그 정리 기능을 설정합니다.
XML:remove_tag,Script:CleanNestedTagOptions.RemoveTag  제거할 태그명을 설정합니다.
 XML:nested_count,Script:CleanNestedTagOptions.NestedCount  중복된 태그 제거 기준 개수를 설정합니다.

remarks

XML:remove_tag,Script:CleanNestedTagOptions.RemoveTag  제거할 태그명을 콤마(,)로 구분하여 입력합니다.
 XML:nested_count,Script:CleanNestedTagOptions.NestedCount  기본값은 "50" 이며 중복된 태그 제거 기준 개수를 설정합니다.

sample code

p,div 태그를 30개 이상 중복된 태그가 있을 경우 제거하도록 설정합니다.

XML

.....
<setting>
    .....
    <clean_nested_tag_options>
        <remove_tag>p,div</remove_tag>    
        
        <nested_count>30</nested_count>    
    </clean_nested_tag_options>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkeditor/js/raonkeditor.js"></script>

<!-- ..... 생략 ..... -->
