K Editor :: OpenDocument

XML : <word max_size="">.....</word>, Script : Word.MaxSize
XML : <word max_page="">.....</word>, Script : Word.MaxPage
XML : <excel max_size="">.....</excel>, Script : Excel.MaxSize
XML : <excel max_sheet="">.....</excel>, Script : Excel.MaxSheet
XML : <powerpoint max_size="">.....</powerpoint>, Script : Powerpoint.MaxSize
XML : <powerpoint max_slide="">.....</powerpoint>, Script : Powerpoint.MaxSlide
XML : <open_document use_hwp="">.....</open_document>, Script : OpenDocument.UseHwp
XML : <open_document use_html5_fileopen="">.....</open_document>, Script : OpenDocument.UseHtml5FileOpen

Script 설정이 우선 적용 됩니다.
XML : max_size, Script : MaxSize  파일 열기할 word, powerpoint, excel파일의 최대 용량을 설정합니다.
XML : max_page, Script : MaxPage  파일 열기할 word파일의 최대 page 수를 설정합니다.
XML : max_sheet, Script : MaxSheet  파일 열기할 excel파일의 최대 sheet 수를 설정합니다.
XML : max_slide, Script : MaxSlide  파일 열기할 powerpoint파일의 최대 slide 수를 설정합니다.
XML : use_hwp, Script : UseHwp  파일 열기기능 사용시 hwp파일 open 가능 여부를 설정합니다.
XML : use_html5_fileopen, UseHtml5FileOpen : Value  파일 열기기능 사용시 html파일 open 가능 여부를 설정합니다.

remarks

XML : use_hwp, Script : UseHwp  "0"으로 설정시 파일 열기기능 사용시 hwp파일 open 불가능, "1"로 설정시 가능하도록 설정합니다.
XML : use_html5_fileopen, UseHtml5FileOpen : Value  "0"으로 설정시 파일 열기기능 사용시 html파일 open 불가능, "1"로 설정시 가능하도록 설정합니다.

sample code

파일 열기 기능을 설정합니다.

XML

.....
<setting>
    .....
    
    <open_document use_hwp="1" use_html5_fileopen="1">
        <max_size="10000", word max_page="2"><word>
        <max_size="10000", excel max_sheet="2"><excel>
        <powerpoint max_size="10000", max_slide="2"><powerpoint>
    </open_document>
    .....
</setting>
.....
