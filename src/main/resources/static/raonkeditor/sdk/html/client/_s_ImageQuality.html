K Editor :: ImageQuality

XML : <image_quality>.....</image_quality>, Script : ImageQuality.Quality
XML : <image_quality worker_count="....."></image_quality>, Script : ImageQuality.WorkerCount
XML : <image_quality extension="....."></image_quality>, Script : ImageQuality.Extension
XML : <image_quality over_file_size="....."></image_quality>, Script : ImageQuality.OverFileSize

Script 설정이 우선 적용 됩니다.

서버에 업로드되는 이미지의 품질을 설정 합니다.

remarks

Quality  기본값은 "1"이고 0.0 ~ 1.0까지 비율로 품질을 낮춥니다.

worker_count  브라우저의 Worker사용 개수를 설정합니다.
    기본값은 7 입니다.
    파일 개수가 Worker 생성 개수보다 작은 경우는 파일 개수만큼만 Worker를 생성합니다.

extension  적용할 확장자를 설정합니다.
콤마(',')로 구분합니다.
기본값은 "" 이며 모든 이미지 파일 확장자에 적용됩니다.

over_file_size  특정 사이즈 이상인 경우에만 품질을 낮춥니다.
파일단위를 포함한 값을 입력합니다.
기본값은 "0" 이며 모든 이미지 파일 사이즈에 적용됩니다.

sample code

1MB 이상의 jpg,png 파일의 이미지 품질을 50%(0.5)로 낮춥니다.
Worker 사용시 최대 5개의 Worker를 생성합니다.

XML

.....
<setting>
    .....
    <image_quality worker_count="5" extension="png,jpg" over_file_size="1MB">0.5</image_quality>
    .....
</setting>
.....
