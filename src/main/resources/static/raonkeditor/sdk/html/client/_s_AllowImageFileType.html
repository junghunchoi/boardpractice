K Editor :: AllowImageFileType

XML : <allow_image_file_type>.....</allow_image_file_type>, Script : AllowImageFileType
XML : <allow_image_file_type max_file_size="....."></allow_image_file_type>, Script : MaxImageFileSize

Script 설정이 우선 적용됩니다.
이미지 삽입 시 허용할 확장자를 설정합니다.
이미지 삽입 시 허용할 최대 파일 사이즈를 제한합니다.

remarks

XML : allow_image_file_type, Script : AllowImageFileType   확장자를 ","(콤마)로 구분하여 입력합니다.
XML : max_file_size, Script : MaxImageFileSize   용량을 바이트단위까지 입력합니다.

sample code

이미지 삽입 시 허용할 확장자를 "jpg,png"로 설정합니다.
이미지 삽입 시 허용할 최대 파일 사이즈를 "5MB"로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <allow_image_file_type max_file_size="5MB">jpg,png</allow_image_file_type>
    .....
</uploader_setting>
.....
