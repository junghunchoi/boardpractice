K Upload :: SkipSentFile

XML : <skip_sent_file>.....</skip_sent_file>, Script : SkipSentFile

Script 설정이 우선 적용됩니다.
기존에 업로드 되었던 파일을 제외하고 업로드하도록 설정합니다.
업로드 완료된 파일이 리스트에 남아있을시 적용됩니다.

remarks

"0" 으로 설정 시 기존에 업로드 되었던 파일을 포함하여 업로드 하고,
"1" 로 설정 시 기존에 업로드 되었던 파일을 제외하고 업로드 합니다.

sample code

기존에 업로드 되었던 파일을 제외하고 업로드하도록 설정합니다.

XML

.....
<setting>
    .....
    <skip_sent_file>1</skip_sent_file>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            SkipSentFile: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
