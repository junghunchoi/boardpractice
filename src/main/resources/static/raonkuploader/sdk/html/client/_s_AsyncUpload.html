K Upload :: AsyncUpload

XML : <async_upload>.....</async_upload>, Script : AsyncUpload

Script 설정이 우선 적용됩니다.
업로드 시 XMLHttpRequest의 async를 설정합니다.

remarks

기본값은 XMLHttpRequest가 동기식(async:true)으로 동작합니다.
설정값 "1"로 설정 시 업로드시 XMLHttpRequest가 동기식(async:true)으로 동작 하고,
"0" 으로 설정 시 업로드시 XMLHttpRequest가 비동기식(async:false)으로 동작합니다.

sample code

업로드시 XMLHttpRequest가 비동기식(async:false)으로 동작하도록 설정합니다.

XML

.....
<uploader_setting>
    .....
    <async_upload>0</async_upload>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            AsyncUpload: "0"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
