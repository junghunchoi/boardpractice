[하이브리드 전용]

K Upload :: DownloadHybridApp

XML : <download_hybrid_app>.....</download_hybrid_app>, Script : DownloadHybridApp

Script 설정이 우선 적용됩니다.
안드로이드 하이브리드앱 관련설정을 적용합니다.

remarks

"0" 으로 적용하지 않고,
"1" 로 설정시 체크박스 제거, 다운로드 방식 변경(A태그로 다운로드) 등을 적용합니다.

sample code

안드로이드 하이브리드앱 관련설정을 적용합니다.

XML

.....
<setting>
    .....
    <download_hybrid_app>1</download_hybrid_app>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            DownloadHybridApp: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
