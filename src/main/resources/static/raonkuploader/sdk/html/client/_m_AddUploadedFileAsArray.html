K Upload :: AddUploadedFileAsArray

void AddUploadedFileAsArray(FileUniqueKey, FileName, FilePath, FileSize, CustomValue, UploadId)

업로드 리스트 컨트롤에 이미 업로드 되어 있는 파일들을 셋팅합니다.

return value

없음.

parameters

FileUniqueKey  추가 할 파일들의 Unique Key을 포함한 Array를 의미합니다.
FileName  추가 할 파일들의 이름을 포함한 Array를 의미합니다.
FilePath  추가 할 파일들의 저장 위치를 포함한 Array를 의미합니다.(파일의 물리적 경로 또는 가상 경로로 설정)
FileSize  추가 할 파일들의 크기를 포함한 Array를 의미합니다.
CustomValue  Custom Download 사용시 사용될 값을 포함한 Array를 의미합니다.
UploadId  파일들을 추가 할 업로드 객체의 id를 의미합니다.

remarks

추가하려는 웹파일 여러개를 배열로 한번에 추가할 수 있습니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">
        function RAONKUPLOAD_CreationComplete(uploadID) {
            // 업로드 생성 후 업로드 리스트 컨트롤에 웹파일들을 배열로 추가
            // 배열에 각 파일의 순서별로 해당 파일의 정보를 입력합니다.
            var FileUniqueKeyArray = [.....];
            var FileNameArray = [.....];
            var FilePathArray = [.....];
            var FileSizeArray = [.....];
            var CustomValueArray = [.....];
            RAONKUPLOAD.AddUploadedFileAsArray(FileUniqueKeyArray, FileNameArray, FilePathArray, FileSizeArray, CustomValueArray, 'upload1');
        }
</script>
