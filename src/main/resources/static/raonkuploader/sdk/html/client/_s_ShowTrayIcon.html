K Upload :: ShowTrayIcon

XML : <show_tray_icon>.....</show_tray_icon>, Script : ShowTrayIcon

Script 설정이 우선 적용됩니다.
Agent 동작시 작업표시줄의 트레이 아이콘이 보이도록 또는 숨기도록 설정합니다.

remarks

"0" 으로 설정 시 트레이 아이콘을 숨기도록 설정하고,
"1" 로 설정 시 트레이 아이콘을 보이도록 설정합니다.

sample code

Agent 동작시 작업표시줄의 트레이 아이콘을 보이도록 설정합니다.

XML

.....
<setting>
    .....
    <show_tray_icon>1</show_tray_icon>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ShowTrayIcon: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
