K Upload :: ChangeToAgentMode

void ChangeToAgentMode(uploadID, skipFileDeleteEvent)

해당 ID의 업로드를 Destroy하고 Agent 모드로 새로 업로드를 로드합니다.

return value

없음.

parameters

uploadID  전송을 취소할 업로드의 id를 의미합니다.
skipFileDeleteEvent  기존에 리스트에 추가된 파일들의 삭제 동작의 수행/스킵 여부를 의미합니다. (true:삭제동작 건너뜀, false:삭제동작 수행)

remarks

없음.

<script type="text/javascript">
        function ChangeToAgentMode() {
            RAONKUPLOAD.ChangeToAgentMode("upload1", true);
        }
</script>
