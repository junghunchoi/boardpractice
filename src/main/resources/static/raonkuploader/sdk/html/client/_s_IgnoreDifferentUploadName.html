K Upload :: IgnoreDifferentUploadName

XML : <ignore_different_upload_name>.....</ignore_different_upload_name>,
Script : IgnoreDifferentUploadName

Script 설정이 우선 적용됩니다.
업로드를 하나만 생성한 이후 API 사용중 아이디를 잘못 넣을 경우 API Call 오류가 발생하지 않도록 설정합니다.

remarks

기본값은 "0" 이며 사용 안함으로 설정하고,
"1"로 설정 시 업로드를 하나만 생성한 이후 API 사용중 아이디를 잘못 넣을 경우 API Call 오류가 발생하지 않도록 설정합니다.

sample code
