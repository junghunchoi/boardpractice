[Agent 전용]

K Upload :: AgentEventType

XML : <agent_event_type>.....</agent_event_type>, Script : AgentEventType

Script 설정이 우선 적용됩니다.
이벤트 폴링 방식을 설정합니다.

remarks

"0" 으로 설정시 특정 interval 기간(AgentEventInterval에 설정된 값)으로 폴링하도록 설정하고,
"1" 로 설정시 특정 시간(AgentEventWaitTime에 설정된 값)안에 결과가 생기면 바로 리턴합니다.

sample code

이벤트 폴링 방식을 interval 방식으로 설정합니다.

XML

.....
<setting>
    .....
    <agent_event_type>0</agent_event_type>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            AgentEventType: "0"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
