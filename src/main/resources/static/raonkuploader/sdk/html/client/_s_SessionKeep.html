K Upload :: SessionKeep

XML : <use_session_keep>.....</use_session_keep>, Script : SessionKeep

Script 설정이 우선 적용됩니다.
WEB,WAS 분리된 환경에서 WEB Server의 Session 유지를 위한 주기적 Request의 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 설정 시 기능을 사용하지 않고,
"1" 으로 설정 시 WEB,WAS 분리된 환경에서 WEB Server의 Session 유지를 위한 주기적으로 Request하도록 설정합니다.

sample code

WEB,WAS 분리된 환경에서 WEB Server의 Session 유지를 위한 주기적으로 Request하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_session_keep>1</use_session_keep>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            SessionKeep: {
                Use: "1"
            }
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
