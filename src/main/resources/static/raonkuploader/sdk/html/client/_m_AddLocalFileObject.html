K Upload :: AddLocalFileObject

void AddLocalFileObject(fileObject, customValue, uploadID)

File 태그와 업로드 연동이 필요한 경우 파일을 추가 하고 전송할 수 있습니다.

return value

없음.

parameters

fileObject  첨부할 파일태그를 의미합니다.
customValue  해당 파일에 설정할 CustomValue 정보를 의미합니다.
uploadID  첨부하려는 업로드의 id값을 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">
        function addLocalFileObject() {
            var fileObject = document.getElementById("form_File");
            RAONKUPLOAD.AddLocalFileObject(fileObject, 'customValue', 'upload1');
        }
</script>
