K Upload :: AgentInstallFailCheck

XML : <agent_install_fail_check>.....</agent_install_fail_check>,
Script : AgentInstallFailCheck.Use
XML : <agent_install_fail_check stop_install_check="....."></agent_install_fail_check>,
Script : AgentInstallFailCheck.StopInstallCheck
XML : <agent_install_fail_check time="....."></agent_install_fail_check>,
Script : AgentInstallFailCheck.Time

Script 설정이 우선 적용됩니다.
Agent 설치의 실패 또는 중단의 체크여부를 설정합니다.

remarks

stop_install_check  false 로 설정 시 AgentInstallFailCheck를 중단하지 않도록 설정하고,
true 로 설정 시 time에 설정된 체크 이후 AgentInstallFailCheck를 중단합니다. time  팝업후 Agent 설치의 실패를 체크하는 시간단위를 설정합니다. (단위: 초)

sample code

팝업 이후 60초 뒤 Agent 설치의 실패를 체크하고 이후 AgentInstallFailCheck를 중단합니다.

XML

.....
<setting>
    .....
    <agent_install_fail_check stop_install_check="1" time="60">1</agent_install_fail_check>
    .....
</setting>
