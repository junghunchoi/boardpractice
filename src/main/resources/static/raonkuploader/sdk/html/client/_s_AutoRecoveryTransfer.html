K Upload :: AutoRecoveryTransfer

XML : <auto_recovery_transfer>.....</auto_recovery_transfer>, Script : AutoRecoveryTransfer
XML : <auto_recovery_transfer auto_transfer="....."></auto_recovery_transfer>, Script : AutoTransfer

Script 설정이 우선 적용됩니다.
자동 복구 업/다운로드 기능을 설정합니다.

remarks

auto_recovery_transfer  "0" 으로 설정시 자동 복구 업/다운로드 사용하지 않고,
"1" 로 설정시 창을 닫거나 OS종료시에도 자동 복구 업/다운로드를 진행하도록 설정하고,
"2" 로 설정시 "1" 설정의 기능에 취소기능을 포함하도록 설정합니다.
auto_transfer  "0" 으로 설정시 업로드 로드 후 자동 복구건이 있어도 즉시 업/다운로드를 진행하지않고,
"1" 로 설정시 업로드 로드 후 자동 복구건이 있으면 즉시 업/다운로드를 진행합니다.
(auto_recovery_transfer 기능을 사용할 시 적용됩니다.)

sample code

자동 복구 업/다운로드 기능을 취소가 가능하도록 사용하고 업로드 후 자동복구건이 존재하면 바로 수행하도록 설정합니다.
