[Agent 전용]

K Upload :: TransferOpenFile

XML : <transfer_open_file>.....</transfer_open_file>, Script : TransferOpenFile

Script 설정이 우선 적용됩니다.
전송하려는 파일이 열려있을 때 쓰기,수정 권한을 통해 전송 허용 옵션을 설정합니다.

remarks

"0" 으로 설정 시 agent 모듈이 해당 파일 업로드 전 파일에 대한 쓰기,수정 권한을 얻지 못하면 업로드 하지 않고 오류 처리를 합니다.
      Ex1) 문서 파일이 열려있는 경우 문서프로그램 프로세스에 의해 파일이 lock이 되어 있어 agent 모듈은 쓰기 권한을 얻지 못할 수 있습니다.
            이 경우에 파일 전송을 하지 않고 오류 처리합니다.
      Ex2) txt 파일이 notepad로 열려 있는 경우 agent는 해당 파일에 대한 쓰기 권한을 얻을 수 있기 때문에 파일을 전송합니다.
"1" 로 설정 시 전송하려는 파일이 열려있을 때 전송을 허용합니다.

sample code

전송하려는 파일이 열려있을 때 전송하도록 설정합니다.

XML

.....
<setting>
    .....
    <transfer_open_file>1</transfer_open_file>
    .....
</setting>
.....
