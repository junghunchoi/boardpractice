K Upload :: UploadSkipErrorFile

XML : <upload_skip_error_file>.....</upload_skip_error_file>, Script : UploadSkipErrorFile

Script 설정이 우선 적용됩니다.
업로드 중 fail된 파일의 정보를 받거나 제외하도록 설정합니다.

remarks

"0" 으로 설정 시 업로드 중 fail된 파일들을 완료 결과값에 포함하고,
"1" 로 설정 시 업로드 중 fail된 파일들을 완료 결과값에 포함하지 않습니다.

sample code

업로드 중 fail된 파일들을 완료 결과값에 포함하지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <upload_skip_error_file>1</upload_skip_error_file>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UploadSkipErrorFile: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
