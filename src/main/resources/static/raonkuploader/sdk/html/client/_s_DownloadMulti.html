[웹표준모드 전용]

K Upload :: DownloadMulti

XML : <download_multi>.....</download_multi>, Script : DownloadMulti

Script 설정이 우선 적용됩니다.
웹표준 모드에서 2개이상의 파일을 다운로드 할때 개별파일로 다운받을지를 설정합니다.

remarks

기본값은 "0" 으로 설정시 2개이상의 파일을 다운로드 받을 경우 압축파일로 다운로드 하도록 설정하고,
"1" 로 설정시 2개이상의 파일을 다운로드 받을 경우 개별파일로 다운로드 하도록 설정합니다.

sample code

2개이상의 파일을 다운로드 받을 경우 개별파일로 다운로드 하도록 설정합니다.

XML

.....
<uploader_setting>
    .....
    <download_multi>1</download_multi>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            DownloadMulti: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
