K Upload :: UseZipDownload

XML : <use_zip_download>.....</use_zip_download>, Script : UseZipDownload

Script 설정이 우선 적용됩니다.
여러 개 파일(2개 이상) 다운로드 시 zip 파일로 다운로드 하도록 설정합니다.

remarks

기본값은 "0" 이며 설정 시 각 파일 그대로 다운로드 하고,
"1" 으로 설정 시 여러 개 파일(2개 이상) 다운로드 시 zip 파일로 다운로드 하도록 설정합니다.

sample code

여러 개 파일(2개 이상) 다운로드 시 zip 파일로 다운로드 하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_zip_download>1</use_zip_download>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseZipDownload: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
