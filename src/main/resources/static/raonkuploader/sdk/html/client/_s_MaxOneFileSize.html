K Upload :: MaxOneFileSize

XML : <max_one_file_size unit=".....">.....</max_one_file_size>,
Script : MaxOneFileSize
XML : <max_one_file_size file_size_by_extension=".....">.....</max_one_file_size>,
Script : MaxOneFileSizeByExtension

Script 설정이 우선 적용됩니다.
XML:max_one_file_size, Script:MaxOneFileSize  업로드 시 파일당 용량 제한을 설정합니다.
XML:file_size_by_extension, Script:MaxOneFileSizeByExtension  업로드 시 파일 확장자별 용량 제한을 설정합니다.

remarks

XML:max_one_file_size  기본 값은 "0" 으로 용량 제한 없음이고, "사용자 설정 값" 으로 설정합니다.
파일 단위는 max_one_file_size의 unit 속성 값에 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
Script:MaxOneFileSize  기본 값은 "0" 으로 용량 제한없음이고, "사용자 설정 값" 으로 설정합니다.
파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
XML:file_size_by_extension  기본 값은 용량 제한 없음이고, "확장자|용량" 으로 설정합니다.
파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.
Script:MaxOneFileSizeByExtension  기본 값은 용량 제한없음이고, "확장자|용량" 으로 설정합니다.
파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.

※ 확장자별 용량 제한 값이 파일당 용량 제한 값보다 클 경우, 파일당 용량 제한 값으로 적용됩니다.

sample code

업로드 시 파일당 최대 크기를 10MB로 제한합니다.
업로드 시 jpg 파일의 최대 크기를 5MB로 제한합니다.

XML

.....
<setting>
    .....
    <max_one_file_size unit="MB" file_size_by_extension="jpg|5MB">10</max_one_file_size>
    .....
</setting>
.....
