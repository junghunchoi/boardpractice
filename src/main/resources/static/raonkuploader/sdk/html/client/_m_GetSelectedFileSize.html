K Upload :: GetSelectedFileSize

void GetSelectedFileSize(uploadID)

업로드 영역에서 선택된 파일 사이즈의 합을 가져옵니다.

return value

int   업로드 파일리스트에 선택된 파일들의 사이즈 합을 의미합니다.

parameters

uploadID  업로드의 id를 의미합니다.

remarks

선택된 항목이 없을 시 -1을 반환합니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function GetSelectedFileSize() {
            var size = RAONKUPLOAD.GetSelectedFileSize("upload1");
        }
</script>
