K Upload :: AddRAONKFileObject

void AddRAONKFileObject(raonkFileObjectList, UploadId)

raonkFileObjectList 안의 파일 오브젝트들(웹파일 및 업로드파일)을 파일 리스트에 추가합니다.

return value

없음.

parameters

raonkFileObjectList  파일오브젝트 및 파일정보를 오브젝트들이 들어간 Array 입니다.
- iswebfile  추가 할 파일이 웹파일인지의 여부를 의미합니다.
- uniqueKey  추가 할 파일의 Unique Key를 의미합니다.
- originalName  추가 할 파일의 이름을 의미합니다.
- uploadPath  추가 할 웹파일이 저장된 위치(파일의 물리적 경로 또는 가상 경로)를 의미합니다. (iswebfile이 1일경우 반드시 필요합니다.)
- size  추가 할 파일의 크기를 의미합니다.
- customValue  Custom Download 사용시 사용될 값을 의미합니다.
- localPath  업로드할 파일의 경로를 의미합니다. (iswebfile이 0일경우 반드시 필요합니다.)
- fileObject  Custom Download 사용시 사용될 값을 의미합니다. (웹표준 모드에서 사용할 경우 반드시 필요합니다.)
UploadId  파일을 추가 할 업로드 객체의 id를 의미합니다.

remarks

웹파일과 업로드할 파일들을 동시에 리스트에 추가 가능합니다.
