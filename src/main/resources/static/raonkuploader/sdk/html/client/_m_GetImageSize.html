K Upload :: GetImageSize

void GetImageSize(itemIndex, callBackFunc, uploadID)

해당 인덱스에 존재하는 이미지파일의 사이즈를 callbackFunc의 파라미터로 반환받습니다.

return value

callBackFunc의 첫번째 파라미터에 return값을 반환받습니다.
object   해당 index의 이미지파일 크기를 포함한 object 입니다.
- object.width  이미지의 가로길이를 의미합니다.
- object.height  이미지의 세로길이를 의미합니다.

parameters

itemIndex  파일리스트의 인덱스를 의미합니다.
callBackFunc  수행할 callBackFunc를 의미합니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.

sample code

        function GetImageSize() {
            var callBackFunc = function(imageSizeObj){

            }
            RAONKUPLOAD.GetImageSize(....., callBackFunc, "upload1");
        }
</script>
