K Upload :: UseDropZoneOverClass

XML : <use_dropzone_over_class>.....</use_dropzone_over_class>, Script : UseDropZoneOverClass

Script 설정이 우선 적용됩니다.
드랍존에 마우스 오버시 클래스명을 "raonkdropzone_over"로 변경 여부를 설정합니다.

remarks

기본값은 "-1" 이며 설정 시 웹 표준 모드에서만 드랍존에 마우스 오버시 클래스명을 변경하고,
"0" 으로 설정 시 웹 표준모드와 Agent모드 모두 드랍존에 마우스 오버시 클래스명을 변경하지 않고,
"1" 로 설정 시 웹 표준모드와 Agent모드 모두 드랍존에 마우스 오버시 클래스명을 변경하도록 설정합니다.

sample code

웹 표준모드와 Agent모드 모두 드랍존에 마우스 오버시 클래스명을 변경하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_dropzone_over_class>1</use_dropzone_over_class>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseDropZoneOverClass: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
