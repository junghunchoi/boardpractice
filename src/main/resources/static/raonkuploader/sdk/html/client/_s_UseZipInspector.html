K Upload :: UseZipInspector

XML : <use_zip_inspector>.....</use_zip_inspector>, Script : UseZipInspector

Script 설정이 우선 적용됩니다.
압축파일 보기(zip 탐색) 기능의 사용여부를 설정합니다.
설정 시 웹파일인 압축파일을 우클릭하여 zip 탐색이 가능합니다.

remarks

기본값은 "0" 이며 설정 시 zip 탐색 기능을 사용하지 않고,
"1" 로 설정 시 zip 탐색 기능을 사용하도록 설정합니다.

sample code

웹파일인 압축파일을 우클릭하여 zip 탐색이 가능하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_zip_inspector>1</use_zip_inspector>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseZipInspector: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
