K Upload :: AgentEventWaitTime

XML : <agent_event_wait_time>.....</agent_event_wait_time>, Script : AgentEventWaitTime

Script 설정이 우선 적용됩니다.
AgentEventType 설정값이 1일 때 Wait 시간을 설정합니다.

remarks

AgentEventType 설정값이 1일 때 적용되며 단위는 분 입니다.

sample code

이벤트 리턴 Wait 시간을 2분으로 설정합니다.

.....
<setting>
    .....
    <agent_event_wait_time>2</agent_event_wait_time>
    .....
</setting>
.....
