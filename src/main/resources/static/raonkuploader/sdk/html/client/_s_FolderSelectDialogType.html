K Upload :: FolderSelectDialogType

XML : <folder_select_dialog_type>.....</folder_select_dialog_type>,
Script : FolderSelectDialogType

Script 설정이 우선 적용됩니다.
폴더 추가 dialog의 type을 설정합니다.

remarks

"0" 으로 설정시 폴더 추가 dialog를 일반 탐색기 dialog로 설정하고,
"1" 로 설정시 폴더 추가 dialog를 간소화된 리스트형 dialog로 설정합니다.

sample code

폴더 추가 dialog를 간소화된 리스트형 dialog로 설정합니다.

XML

.....
<setting>
    .....
    <folder_select_dialog_type>1</folder_select_dialog_type>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            FolderSelectDialogType: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
