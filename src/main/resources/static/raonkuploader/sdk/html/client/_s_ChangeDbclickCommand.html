K Upload :: ChangeDbclickCommand

XML : <change_dbclick_command>.....</change_dbclick_command>,
Script : ChangeDbclickCommand

Script 설정이 우선 적용됩니다.
리스트의 웹파일 더블클릭시의 동작을 설정합니다.

remarks

미 설정시 기본값인 열기동작이 수행되며 열기 확장자를 설정한 경우 해당 확장자 파일에만 적용됩니다.
"download" 로 설정시 더블클릭시 파일을 다운로드 하도록 설정하고,
"downloadandopen" 로 설정시 더블클릭시 파일을 다운로드 후 열도록 설정하고,
"downloadandfolderopen" 로 설정시 더블클릭시 파일을 다운로드 후 다운로드 위치의 폴더를 열도록 설정합니다.

sample code

더블클릭시 파일을 다운로드 후 다운로드 위치의 폴더를 열도록 설정합니다.

XML

.....
<setting>
    .....
    <change_dbclick_command>downloadandfolderopen</change_dbclick_command>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->

    </script>
