K Upload :: FileSort

void FileSort(sortType, sortMethod, uploadID)

파일 리스트의 파일들을 정렬합니다.

return value

없음.

parameters

sortType  정렬 기준을 의미합니다. (0:이름순, 1:크기순, 2:headerEx순)
sortMethod  정렬 방식을 의미합니다. (0:오름차순, 1:내림차순)
uploadID  리스트의 파일들을 정렬할 업로드의 id를 의미합니다.

remarks

없음.
