K Upload :: UseInstallGuide

XML : <use_install_guide>.....</use_install_guide>,
Script : UseInstallGuide
XML : <use_install_guide logo_url="....."></use_install_guide>,
Script : InstallGuideLogoUrl
XML : <use_install_guide zindex="....."></use_install_guide>,
Script : InstallGuideZIndex

Script 설정이 우선 적용됩니다.
Agent 설치 시 가이드창의 사용 여부를 설정합니다.
Agent 설치 가이드창을 설정합니다.

remarks

use_install_guide  기본값은 "0" 으로 설정 시 Agent 설치시 설치 안내창을 노출하지 않고, "1"은 노출하도록 설정합니다.
logo_url  Agent 설치 안내창의 로고 이미지 url을 설정합니다. (기본값: "images/BI-raonk.png")
zindex  Agent 설치 안내창에 대한 ZIndex를 설정합니다. (기본값: 1000)

sample code

Agent 설치시 설치 안내창 노출여부를 노출로 설정합니다.
Agent 설치 안내창의 로고 이미지 url을 설정합니다.
Agent 설치 안내창의 ZIndex는 10000으로 설정합니다.

.....
<setting>
    .....
    <use_install_guide logo_url="....." zindex="10000">1</use_install_guide>
    .....
</setting>
