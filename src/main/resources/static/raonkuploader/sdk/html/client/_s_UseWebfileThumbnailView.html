K Upload :: UseWebfileThumbnailView

XML : <use_webfile_thumbnail_view>.....</use_webfile_thumbnail_view>,
Script : UseWebfileThumbnailView

Script 설정이 우선 적용됩니다.
Thumbnail 보기 모드일 때 웹파일의 경우 Thumbnail 표시여부를 설정합니다.

remarks

기본값은 "1" 이며 설정 시 Thumbnail 보기 모드일 때 웹파일의 경우에도 Thumbnail을 표시하고,
"0" 으로 설정 시 Thumbnail 보기 모드일 때 웹파일의 경우 Thumbnail을 표시하지 않습니다.

sample code

Thumbnail 보기 모드일 때 웹파일의 경우 Thumbnail을 표시하지 않습니다.

XML

.....
<setting>
    .....
    <use_webfile_thumbnail_view>1</use_webfile_thumbnail_view>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseWebfileThumbnailView: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
