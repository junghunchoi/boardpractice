K Upload :: ThumbsSize

XML : <thumbs_size>.....</thumbs_size>, Script : ThumbsSize

Script 설정이 우선 적용됩니다.
썸네일 보기 모드(업로드의 views 모드가 thumbs)시 적용됩니다.
업로드 리스트의 썸네일 크기를 설정합니다.

remarks

썸네일의 width, height의 기본값은 각각 88, 60 이며, 이를 초과한 값 입력 시 적용됩니다.
"가로값, 세로값" 으로 입력합니다..

sample code

업로드 리스트 썸네일의 width을 200px, height을 100px로 설정합니다.

XML

.....
<setting>
    .....
    <thumbs_size>200,100</thumbs_size>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ThumbsSize: "200,100"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
