K Upload :: UseAddLocalFileDirectlyCallBackEvent

XML : <use_addlocalfiledirectly_callback_event>.....</use_addlocalfiledirectly_callback_event>, Script : UseAddLocalFileDirectlyCallBackEvent

Script 설정이 우선 적용됩니다.
AddLocalFileDirectly API 사용 시 CallBack 이벤트의 발생여부를 설정합니다.

remarks

"0" 으로 설정 시 이벤트를 설정하여도 발생하지 않도록 설정하고,
"1" 로 설정 시 이벤트를 설정할경우 해당 이벤트가 발생하도록 설정합니다.
넥사크로 전용 브라우저에서 AfterAddAllFile 이벤트를 발생하고 싶을 때 '0'으로 설정합니다.

sample code

AddLocalFileDirectly CallBack 이벤트가 발생하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_addlocalfiledirectly_callback_event>1</use_addlocalfiledirectly_callback_event>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseAddLocalFileDirectlyCallBackEvent: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
