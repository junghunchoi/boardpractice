K Upload :: MoveMouseDrag

XML : <move_mouse_drag>.....</move_mouse_drag>, Script : MoveMouseDrag

Script 설정이 우선 적용됩니다.
파일리스트의 파일을 마우스 드래그시 파일순서의 이동 가능 여부를 설정합니다.

remarks

기본값은 "0" 이며 설정 시 파일리스트의 파일을 마우스 드래그로 순서 변경을 하지 않고,
"1"로 설정 시 파일리스트의 파일을 마우스 드래그로 순서 변경 가능하도록 설정합니다.

sample code

파일리스트의 파일을 마우스 드래그로 순서 변경 가능하도록 설정합니다.

XML

.....
<setting>
    .....
    <move_mouse_drag>1</move_mouse_drag>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            MoveMouseDrag: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
