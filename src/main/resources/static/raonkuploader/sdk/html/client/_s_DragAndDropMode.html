K Upload :: DragAndDropMode

XML : <drag_and_drop_mode>.....</drag_and_drop_mode>, Script : DragAndDropMode

Script 설정이 우선 적용됩니다.
Drop zone(DragAndDrop으로 파일 추가가 가능한 영역)을 브라우저 전체로 설정합니다.

remarks

"0" 으로 설정시 설정하지 않고,
"1" 로 설정시 브라우저 전체를 Drop zone으로 설정합니다.

sample code

브라우저 전체를 Drop zone으로 설정합니다.

XML

.....
<setting>
    .....
    <drag_and_drop_mode>1</drag_and_drop_mode>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            DragAndDropMode: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
