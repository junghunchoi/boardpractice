[Agent 전용]

K Upload :: UseBrowserUseragent

XML : <use_browser_useragent>.....</use_browser_useragent>, Script : UseBrowserUseragent

Script 설정이 우선 적용됩니다.
업/다운로드 시 요청 header에 브라우저 user-agent 값의 추가여부를 설정합니다.

remarks

기본값은 "1" 로 설정시 업/다운로드 시 요청 header에 브라우저 user-agent 값을 추가하도록 설정하고,
"0" 으로 설정시 업/다운로드 시 요청 header에 브라우저 user-agent 값을 추가하지 않도록 설정합니다.

sample code

업/다운로드 시 요청 header에 브라우저 user-agent 값을 추가하지 않도록 설정합니다.

XML

.....
<uploader_setting>
    .....
    <use_browser_useragent>0</use_browser_useragent>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseBrowserUseragent: "0"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
