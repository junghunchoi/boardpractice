K Upload :: HighSpeed

XML : <high_speed>.....</high_speed>, Script : HighSpeed

Script 설정이 우선 적용됩니다.
고속전송모드의 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 설정 시 고속전송모드를 사용하지 않고,
"1" 으로 설정 시 고속전송모드를 사용하도록 설정합니다.

sample code

고속전송모드를 사용하도록 설정합니다.

XML

.....
<uploader_setting>
    .....
    <high_speed>1</high_speed>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            HighSpeed: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
