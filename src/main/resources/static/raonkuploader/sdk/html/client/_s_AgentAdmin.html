Script 설정이 우선 적용됩니다.
업로드/다운로드시 적용될 Admin 설정값을 설정합니다.

remarks

사이즈는 Bytes 단위로 입력합니다.
buffer_size  업로드 및 다운로드 버퍼 사이즈를 설정합니다. (BufferSizeUpload 혹은 BufferSizeDownload 값이 설정될 시 해당값을 우선합니다.)
buffer_size_download  다운로드 시 버퍼 사이즈를 설정합니다.
buffer_size_upload  업로드 시 버퍼 사이즈를 설정합니다.
chunk_size  업로드 및 다운로드 시 청크 사이즈를 설정합니다.
download_file_name_suffix  다운로드 시 파일 확장자 뒤에 임시로 붙히게 될 임시 확장자를 설정합니다.
multi_transfer_count  동시 업로드 및 다운로드 처리 개수를 설정합니다.
request_count  요청(쓰레드)개수를 설정합니다.
system_code  시스템 코드를 설정합니다.
timeout  서버 상태에 대한 응답대기 시간을 설정합니다. (단위: 초)
