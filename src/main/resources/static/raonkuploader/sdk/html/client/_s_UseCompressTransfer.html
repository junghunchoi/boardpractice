K Upload :: UseCompressTransfer

XML : <use_compress_transfer>.....</use_compress_transfer>,
Script : UseCompressTransfer
XML : <use_compress_transfer use_html5="....."></use_compress_transfer>,
Script : UseCompressTransferHtml5

Script 설정이 우선 적용됩니다.
압축전송 옵션의 사용 여부를 설정합니다.

remarks

use_compress_transfer  기본값은 "0" 으로 압축전송 옵션을 사용하지 않고, "1" 로 설정 시 Agent 모드에서 압축전송 옵션을 사용 합니다.
use_html5  기본값은 "0" 으로 웹표준 모드에서 압축전송 옵션을 사용하지 않고, "1" 로 설정 시 웹표준 모드에서 압축전송 옵션을 사용 합니다.
(use_compress_transfer 설정값이 "1" 일때 적용됩니다.)

sample code

Agent 모드와 웹표준 모드에서 압축전송 옵션을 사용하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_compress_transfer use_html5="1">1</use_compress_transfer>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseCompressTransfer: '1',
            UseCompressTransferHtml5: '1'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
