K Upload :: SetMultiFileSelect

void SetMultiFileSelect(boolType, uploadID)

파일추가 버튼을 통해 업로드 리스트에 파일 추가시 다중선택 가능여부를 설정합니다.

return value

없음.

parameters

boolType  true 설정 시 파일추가시 파일 다중선택이 가능하고,
false 설정 시 파일추가시 파일 단일선택으로 적용됩니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function SetMultiFileSelect() {
            RAONKUPLOAD.SetMultiFileSelect(true, "upload1");
        }
</script>
