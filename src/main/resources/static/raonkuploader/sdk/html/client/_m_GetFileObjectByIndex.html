K Upload :: GetFileObjectByIndex

object GetFileObjectByIndex(itemIndex, uploadID)

리스트의 해당 index에 존재하는 파일의 오브젝트를 가져옵니다.

return value

object  해당 인덱스의 파일 오브젝트 입니다.

parameters

itemIndex  파일리스트의 인덱스를 의미합니다.
uploadID  업로드의 id를 의미합니다.

remarks

웹파일은 가져오지 않습니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function GetFileObjectByIndex() {
            var obj = RAONKUPLOAD.GetFileObjectByIndex(....., "upload1");
        }
</script>
