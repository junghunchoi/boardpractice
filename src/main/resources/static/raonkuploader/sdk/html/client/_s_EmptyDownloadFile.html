Script : EmptyDownloadFileText
XML : <text><empty_download_file bold="....."></empty_download_file></text>,
Script : EmptyDownloadFileTextBold
XML : <text><empty_download_file color="....."></empty_download_file></text>,
Script : EmptyDownloadFileTextColor
XML : <text><empty_download_file italic="....."></empty_download_file></text>,
Script : EmptyDownloadFileTextItalic
XML : <text><empty_download_file strike="....."></empty_download_file></text>,
Script : EmptyDownloadFileTextStrike
XML : <text><empty_download_file underline="....."></empty_download_file></text>,
Script : EmptyDownloadFileTextUnderline

Script 설정이 우선 적용됩니다.
업로드가 보기모드(view) 일 경우에 표시됩니다.
다운로드할 파일이 없는 경우 파일리스트 영역에 보여줄 메시지및 해당 메시지의 style을 설정합니다.

remarks

empty_download_file  다운로드할 파일이 없는 경우 파일리스트 영역에 보여줄 메시지를 텍스트를 입력하여 설정합니다.
 bold  메시지 style의 bold적용 여부를 설정합니다. (0:설정안함, 1:설정함)
 color  메시지 색상(색상명, RGB, HEX)을 설정합니다.
italic  메시지 style의 italic적용 여부를 설정합니다. (0:설정안함, 1:설정함)
 strike  메시지 style의 취소선 적용 여부를 설정합니다. (0:설정안함, 1:설정함)
 underline  메시지 style의 밑줄 적용 여부를 설정합니다. (0:설정안함, 1:설정함)

sample code

다운로드할 파일이 없는 경우 파일리스트 영역에 보여줄 메시지및 해당 메시지의 style을 설정합니다.

XML

.....
<setting>
    .....
    <text>
        <empty_download_file bold="1" color="rgb(50,140,200)" italic="1" strike="0" underline="1">File List is Empty.</empty_download_file>
    </text>
    .....
</setting>
.....
