K Upload :: UseWorkingImg.UseDownloadPre

XML : <use_working_image><download_pre>.....</download_pre></use_working_image>,
Script : UseWorkingImg.UseDownloadPre
XML : <use_working_image><download_pre caption="....."></download_pre></use_working_image>,
Script : UseWorkingImg.DownloadPreCaption
XML : <use_working_image><download_pre image="....."></download_pre></use_working_image>,
Script : UseWorkingImg.DownloadPreImage
XML : <use_working_image><download_pre use_once="....."></download_pre></use_working_image>,
Script : UseWorkingImg.DownloadPreUseOnce

Script 설정이 우선 적용됩니다.
다운로드 pre request에서 작업 이미지 관련내용을 설정합니다.

remarks

download_pre  기본값은 "0" 으로 다운로드 pre request에서 작업 이미지를 띄우지 않고, "1" 로 설정 시 pre request에서 작업 이미지를 띄웁니다.
caption  작업 이미지의 캡션을 설정 합니다.
image  작업 이미지 파일의 경로를 설정 합니다.
use_once  기본값은 "0" 으로 작업 이미지를 여러번 띄우고, "1" 로 설정 시 작업 이미지를 한번만 띄웁니다.

sample code

다운로드 pre request에서 작업 이미지 관련내용을 설정합니다.

XML

.....
<setting>
    .....
    <use_working_image>
        <download_pre caption="....." image="....." use_once="1">1<download_pre>
    </use_working_image>
    .....
</setting>
.....

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
