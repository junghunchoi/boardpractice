K Upload :: IsUpdateRequired

void IsUpdateRequired(callBackFunc, mode, versionUrl)

Agent의 업데이트가 필요한 상태인지 확인합니다.

return value

callBackFunc의 첫번째 파라미터에 return값을 반환받습니다.
boolean  Agent의 업데이트가 필요한 상태인지를 의미합니다.

parameters

callBackFunc  실행할 callback 함수를 의미합니다..
mode  0 입력시 PC에 설치된 버전이 최신 버전보다 낮으면 true 반환하고,
1 입력시 PC에 설치된 버전이 최신 버전과 다르면 true를 반환합니다.
versionUrl  Agent 폴더의 경로를 의미합니다. (Agent 폴더의 위치 변경시 파라미터에 해당 경로를 입력해주시면 됩니다.)

remarks

없음.
