K Upload :: SkipInternetState

XML : <skip_internet_state>.....</skip_internet_state>, Script : SkipInternetState

Script 설정이 우선 적용됩니다.
인터넷 환경이 정상인지 체크하는 옵션입니다.

remarks

egg, 핫스팟 등을 주로 사용하는 환경에서는 해당 설정값을 1로 설정하여 써야할 수 있습니다.
"0" 으로 설정 시 인터넷 환경이 정상인지 체크하지않고,
"1" 로 설정 시 인터넷 환경이 정상인지 체크합니다.

sample code

인터넷 환경이 정상인지 체크하지않도록 설정합니다.

XML

.....
<setting>
    .....
    <skip_internet_state>0</skip_internet_state>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->

        }

    </script>
</div>
