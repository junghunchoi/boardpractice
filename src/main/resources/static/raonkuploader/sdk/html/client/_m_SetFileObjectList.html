K Upload :: SetFileObjectList

void SetFileObjectList(fileListObject, uploadID)

Array 안의 오브젝트에 포함된 경로에 있는 파일들의 정보를 읽어 업로드 컴포넌트의 파일 리스트에 추가합니다.

return value

없음.

parameters

fileListObject  파일 경로정보를 포함한 object들을 포함한 Array를 의미합니다.
- object.file  파일의 실제경로(물리적경로)를 의미하며 fileListObject 내의 오브젝트에 포함되는 값입니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function SetFileObjectList() {
            var fileListObject = [
                {file : "....."},
                {file : "....."},
                {file : "....."},
                {file : "....."},
                {file : "....."}
            ]
                
            RAONKUPLOAD.SetFileObjectList(fileListObject, "upload1");
        }
</script>
