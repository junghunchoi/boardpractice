K Upload :: UseAfterMoveFileEvent

XML : <use_after_move_file_event>.....</use_after_move_file_event>, Script : UseAfterMoveFileEvent

Script 설정이 우선 적용됩니다.
추가된 파일의 위치변경 완료 시 이벤트(RAONKUPLOAD_AfterMoveFile)의 발생여부를 설정합니다.

remarks

"0" 으로 설정 시 이벤트를 설정하여도 발생하지 않도록 설정하고,
"1" 로 설정 시 이벤트를 설정할경우 해당 이벤트가 발생하도록 설정합니다.

sample code

RAONKUPLOAD_AfterMoveFile 이벤트가 발생하도록 설정합니다.

XML

.....
<setting>
    .....
    <use_after_move_file_event>1</use_after_move_file_event>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseAfterMoveFileEvent: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
