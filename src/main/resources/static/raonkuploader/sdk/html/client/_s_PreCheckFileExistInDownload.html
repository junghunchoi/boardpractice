K Upload :: PreCheckFileExistInDownload

XML : <pre_check_file_exist_in_download>.....</pre_check_file_exist_in_download>, Script : PreCheckFileExistInDownload

Script 설정이 우선 적용됩니다.
다운로드 시 동일파일 유무 판단의 시점을 다운로드 시작 전 또는 다운로드 후 로 설정합니다.

remarks

기본값은 "0" 이며 설정 시 다운로드 시 동일파일 유무 판단을 다운로드 시작 후 하도록 설정하고,
"1"로 설정 시 다운로드 시 동일파일 유무 판단을 다운로드 시작 전에 하도록 설정합니다.

sample code
