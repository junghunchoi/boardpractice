[Agent 전용]

K Upload :: EnableAdminLog

XML : <enable_admin_log>.....</enable_admin_log>, Script : EnableAdminLog

Script 설정이 우선 적용됩니다.
Agent Admin Log 활성화 여부를 설정합니다.

remarks

"0" 으로 설정시 Agent Admin Log를 비활성화 하고,
"1" 로 설정시 Agent Admin Log를 활성화 합니다.

sample code

Agent Admin Log를 활성화 합니다.

XML

.....
<setting>
    .....
    <enable_admin_log>1</enable_admin_log>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            EnableAdminLog: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
