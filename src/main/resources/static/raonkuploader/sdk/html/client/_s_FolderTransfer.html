K Upload :: FolderTransfer

XML : <folder_transfer>.....</folder_transfer>,
Script : FolderTransfer
XML : <folder_transfer get_folder_in_file="....."></folder_transfer>,
Script : GetFolderInFile

Script 설정이 우선 적용됩니다.
폴더 구조로 업로드를 설정합니다.
폴더 상태로 폴더를 추가, 폴더 안에 있는 파일들을 한번에 추가 합니다.
폴더 추가시 폴더내 파일리스트의 추출여부를 설정합니다.

remarks

folder_transfer  기본값은 "0" 이고, "1"로 설정시 폴더 구조로 업로드 설정합니다.
get_folder_in_file  기본값은 "0" 이고, "1"로 설정시 폴더 추가시 폴더내 파일리스트를 추출합니다.

sample code

폴더 구조로 업로드를 설정합니다.
폴더 추가시 폴더내 파일리스트를 추출합니다.
