K Upload :: AddLocalFolderDirectlyExWithCallback

void AddLocalFolderDirectlyExWithCallback(options, callback, uploadID)

로컬폴더를 업로드 영역에 추가합니다.
성공여부를 callback으로 받습니다.

return value

없음.

parameters

options  로컬폴더의 경로 정보를 담고 있는 object 변수 입니다. (상세내용은 하단 sample code를 참고하세요.)
callback  성공여부를 리턴받는 callback function 입니다. (상세내용은 하단 sample code를 참고하세요.)
uploadID  실행할 업로드의 id를 의미합니다.

remarks

폴더전송 기능을 사용함으로 설정한 경우에만 동작합니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">
        function addLocalFolders() {
            var callback = function (failed) {
                if (failed) { // 실패한 파일이 있는 경우 (에러 또는 용량 제한 등의 필터링)
                    console.log(failed);
                    /*
                        failed.data : array
                        failed.data = [
                            {
                                type: string, -> 'error': 오류인 경우, 'filtered': 사이즈, 확장자 제한 등으로 필터링된 경우
                                index: number, -> 실패된 폴더의 index
                                reason: string -> 실패 원인
                            },
                            ...
                        ]

                        failed.data.length : 실패한 폴더 개수
                    */
                } else { // failed 변수가 null 이면 성공
                    console.log('success');
                }
            };

            var fileDataArray = [
                {
                    path: 'c:\\testfolder1\\'
                },
                {
                    path: 'c:\\testfolder2\\'
                },
                ...
            ];

            var options = {
                addFiles: false, // 생략가능(기본값: false), true로 적용 시 폴더안의 파일들이 컴포넌트에 추가 됩니다.
                folderDataArray: folderDataArray
            };

            RAONKUPLOAD.AddLocalFolderDirectlyExWithCallback(options, callback, 'upload1');
        }
</script>
