K Upload :: FileDelimiter

XML : <file_delimiter unit_attribute_delimiter="....."></file_delimiter>Script : UnitAttributeDelimiter
XML : <file_delimiter unit_delimiter="....."></file_delimiter>Script : UnitDelimiter

Script 설정이 우선 적용됩니다.
return과 parameter의 구분문자를 설정합니다.

remarks

unit_attribute_delimiter  기본값은 \u000C이며 return의 구분문자로 설정합니다.
unit_delimiter  기본값은 \u000B이며 parameter의 구분문자로 설정합니다.

sample code

return과 parameter의 구분문자를 설정합니다.

<setting>
    .....
    <file_delimiter unit_attribute_delimiter="....." unit_delimiter="....."></file_delimiter>
    .....
