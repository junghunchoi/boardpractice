K Upload :: UseAutoHeight

XML : <use_auto_height>.....</use_auto_height>, Script : UseAutoHeight

Script 설정이 우선 적용됩니다.
파일이 리스트에 추가될 때 설정된 업로드 Height보다 파일리스트의 높이가 더 커지면 자동으로 맞추도록 설정합니다.

remarks

기본값은 "0" 이며 설정 시 업로드를 자동으로 늘리지 않도록 설정하고,
"1" 로 설정 시 업로드를 자동으로 늘리도록 설정합니다.

sample code

파일이 리스트에 추가될 때 설정된 업로드 Height보다 파일리스트의 높이가 더 커지면 자동으로 맞추도록 설정합니다.

XML

.....
<setting>
    .....
    <use_auto_height>1</use_auto_height>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseAutoHeight: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
