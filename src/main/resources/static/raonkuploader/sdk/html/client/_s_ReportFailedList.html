K Upload :: ReportFailedList

XML : <report_failed_list>.....</report_failed_list>, Script : ReportFailedList

Script 설정이 우선 적용됩니다.
다운로드 실패한 리스트를 포함한 txt파일의 생성여부를 설정합니다.

remarks

기본값은 "0" 이며 설정 시 다운로드 실패시 실패파일 리스트를 txt파일로 생성하지 않고,
"1"로 설정 시 다운로드 실패시 사용자 경로 선택 후 실패파일 리스트를 해당 경로에 txt파일로 생성하고,
"2"로 설정 시 다운로드 실패시 다운로드 폴더에 자동으로 실패파일 리스트를 txt파일로 생성합니다.

sample code

다운로드 실패시 다운로드 폴더에 자동으로 실패파일 리스트를 txt파일로 생성하도록 설정합니다.

XML

.....
<setting>
    .....
    <report_failed_list>2</report_failed_list>
    .....
</setting>
.....
