K Upload :: Event.DownloadCompleteAllFile

Event.DownloadCompleteAllFile

callback 이벤트를 설정합니다.

remarks

업로드의 설정파일(kupload/config/raonkupload.config.xml)의 use_download_complete_all_event 속성값이 1인 경우에 이벤트가 발생합니다.
해당 속성값은 script의 파라미터 오브젝트 내에서 UseDownloadCompleteAllEvent : "1" 로도 설정이 가능합니다.

이벤트 RAONKUPLOAD_DownloadCompleteAllFile를 설정합니다.
파일 다운로드가 모두 끝난 후 이벤트가 발생합니다.
Event Object로 Event들을 모아서 설정이 가능합니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 이벤트발생시 호출할 함수를 설정합니다.
        var eventFunc = function(eventParam1, eventParam2, eventParam3){
            alert("Event!");
            // work something ...
        }
        
        // 파일 다운로드가 모두 끝난 후 이벤트가 발생합니다.
        var raonkParam = {
            /*
            script에서 아래설정(UseDownloadCompleteAllEvent)을 적용하거나,
            업로드 설정파일에서 use_download_complete_all_event 속성값을 적용할 시 이벤트가 동작합니다.
            UseDownloadCompleteAllEvent : "1",
            */
            
            .....
            Event: {
                DownloadCompleteAllFile : eventFunc,
                // Add Another Event Config ...
            }
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
