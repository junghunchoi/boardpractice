K Upload :: GetListInfoByIndex

object GetListInfoByIndex(type, itemIndex, uploadID)

해당 인덱스에 존재하는 파일 정보를 가져옵니다.

return value

Array, Object, xml, text 전송 완료 후 새롭게 업로드 된 파일 리스트를 의미합니다.

parameters

type  리턴받을 형식입니다. ('array', 'json', 'xml', 'text')
itemIndex  파일리스트의 인덱스를 의미합니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.
