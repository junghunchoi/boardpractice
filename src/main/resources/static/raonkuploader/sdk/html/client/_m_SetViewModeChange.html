K Upload :: SetViewModeChange

void SetViewModeChange(mode, uploadID)

업로드 컴포넌트의 보기 모드를 리스트 모드나 썸네일 모드로 변환합니다.

return value

없음.

parameters

mode  "list" 설정 시 업로드 컴포넌트의 보기 모드를 리스트 모드로 설정하고,
"thumbs" 설정 시 썸네일 모드로 변환합니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function SetViewModeChange() {
            RAONKUPLOAD.SetViewModeChange("thumbs", "upload1");
        }
</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px;">  
        <script type="text/javascript">
            new RAONKUpload({Id: 'upload1'});
        </script>
</div>
