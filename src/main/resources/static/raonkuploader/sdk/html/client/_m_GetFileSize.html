K Upload :: GetFileSize

void GetFileSize(webPath, callBackFunc)

파라미터에 입력한 서버 webPath에 해당하는 파일 크기를 callBackFunc의 파라미터로 반환받습니다.

return value

callBackFunc의 첫번째 파라미터에 return값을 반환받습니다.
array   webPath에 해당하는 파일 크기를 포함한 array 입니다.

parameters

webPath  서버에 존재하는 파일의 경로(webpath)를 의미합니다.
callBackFunc  수행할 callBackFunc를 의미합니다.

remarks

없음.

            }
            RAONKUPLOAD.GetFileSize(".....", callBackFunc);
        }
</script>
