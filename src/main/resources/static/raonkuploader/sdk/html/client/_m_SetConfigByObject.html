K Upload :: SetConfigByObject

void SetConfigByObject(configObject, uploadID)

업로드 생성 완료 후 설정값 변경 시 객체 형태로 설정합니다.

return value

없음.

parameters

configObject  업로드 설정값의 객체를 의미합니다. (클라이언트 가이드 > settings 설정값들의 Script 방식으로 객체 생성이 가능합니다.)
uploadID  실행할 업로드의 id를 의미합니다. (단일 업로드일 경우 생략 가능합니다.)

remarks

※ 넥사크로 전용브라우저에서는 Event 객체 사용이 불가능합니다.
업로드 생성 후 환경설정 값 변경에 유용합니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>	
<script type="text/javascript">          
        // 업로드 생성 완료 이벤트에서 사이즈, 버튼 리스트, 확장자 제한을 변경합니다.  
        function RAONKUPLOAD_CreationComplete(uploadID) {
            var configObject = {
                // 사이즈
                Width: '680px',    
                Height: '400px', 
  
                // 버튼 리스트
                ButtonBarEdit: 'add,open,send',
                ButtonBarView: 'download,open',

                // 확장자 제한
                ExtensionAllowOrLimit: '0',
                ExtensionArr: 'txt,gif'
            }
            RAONKUPLOAD.SetConfigByObject(configObject, "upload1");
        }
</script>	

<!-- ..... 생략 ..... -->

<div style="width:800px;height:200px">  
        <script type="text/javascript">
            new RAONKUpload({Id: 'upload1'});
        </script>
</div>
