K Upload :: SizeForChunkUpload

XML : <size_for_chunk_upload>.....</size_for_chunk_upload>, Script : SizeForChunkUpload

Script 설정이 우선 적용됩니다.
분할업로드 시 파일 분할 사이즈를 설정합니다.

remarks

기본값은 "0" 이며 설정 시 분할업로드 시 파일 분할 사이즈를 "50MB"(52428800)로 설정하고,
데이터의 단위까지 입력하거나 숫자(바이트)로 입력하여 설정합니다.

sample code

분할업로드 시 파일 분할 사이즈를 10MB로 설정합니다.

XML

.....
<uploader_setting>
    .....
    <size_for_chunk_upload>10MB</size_for_chunk_upload>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            SizeForChunkUpload: "10MB"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
