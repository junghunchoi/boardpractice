Script 설정이 우선 적용됩니다.
이미지 품질을 낮추는 기능을 설정합니다.

remarks

기본값은 "1"이고 0 ~ 1까지 비율로 품질을 낮춥니다.

worker_count  브라우저의 Worker사용 개수를 설정합니다.
    기본값은 7 입니다.
    파일 개수가 Worker 생성 개수보다 작은 경우는 파일 개수만큼만 Worker를 생성합니다.
extension  적용할 확장자를 설정합니다.
콤마(',')로 구분합니다.
changed_extension  변환할 확장자를 설정합니다.
기본값은 'jpg'입니다.
over_file_size  특정 사이즈 이상인 경우에만 품질을 낮춥니다.
processing_window  이미지 퀄리티 동작시 Processing 노출 위치를 설정합니다.
"0" 으로 설정 시 업로드 영역에 노출하고, "1" 로 설정 시 부모창 영역에 노출합니다.

sample code

이미지 품질을 0.5 비율로 낮추도록 설정합니다.
브라우저의 Worker사용 개수를 10개로 설정합니다.
확장자가 png인 이미지 파일의 품질을 낮추도록 설정합니다.
이미지 파일의 확장자를 jpg로 변경합니다.
20KB 이상의 파일인 경우 품질을 낮추도록 설정합니다.
Processing 노출 위치를 업로드 영역으로 설정합니다.
