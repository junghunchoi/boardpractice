K Upload :: UseWS

XML : <runtimes use_ws="....."></runtimes>, Script : UseWS

Script 설정이 우선 적용됩니다.
html5plus의 사용자가 ieplugin을 설정한 경우 웹표준 동작방식을 결정하는 설정입니다.

remarks

기본값은 "1" 이며 설정 시 ieplugin에서 html5plus로 동작 하도록 설정하고,
"0" 으로 설정 시 ieplugin에서 html5plus로 동작하지 않도록 설정합니다.

sample code

ieplugin에서 html5plus로 동작하지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <runtimes use_ws="0">1</runtimes>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseWS: "0"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
