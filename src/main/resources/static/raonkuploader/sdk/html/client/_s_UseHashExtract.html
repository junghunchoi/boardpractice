K Upload :: UseHashExtract

XML : <use_hash_extract>.....</use_hash_extract>, Script : UseHashExtract

Script 설정이 우선 적용됩니다.
업로드 시 Hash값 추출 기능을 설정합니다.

remarks

기본값은 "0" 이며 설정 시 Hash값을 추출하지 않고,
"1" 으로 설정 시 MD5 방식으로 생성한 Hash값을 추출하고,
"2" 로 설정 시 SHA-256 방식으로 생성한 Hash값을 추출합니다.

sample code

업로드 시 SHA-256 방식으로 생성한 Hash값을 추출합니다.

XML

.....
<setting>
    .....
    <use_hash_extract>2</use_hash_extract>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseHashExtract: "2"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
