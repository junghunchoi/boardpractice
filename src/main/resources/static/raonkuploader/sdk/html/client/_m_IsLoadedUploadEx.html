K Upload :: IsLoadedUploadEx

boolean IsLoadedUploadEx(uploadID)

해당 ID의 업로드가 load완료된 상태인지 확인합니다.

return value

boolean  해당 업로드가 load완료 되었는지를 의미합니다.

parameters

uploadID  업로드의 id를 의미합니다.

remarks

업로드 ID를 입력하지 않을 시 항상 false를 반환합니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function IsLoadedUploadEx() {
            var isLoaded = RAONKUPLOAD.IsLoadedUploadEx("upload1");
        }
</script>
