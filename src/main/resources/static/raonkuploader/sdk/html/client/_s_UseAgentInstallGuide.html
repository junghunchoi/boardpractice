K Upload :: UseAgentInstallGuide

XML : <use_agent_install_guide>.....</use_agent_install_guide>,
Script : UseAgentInstallGuide
XML : <use_agent_install_guide zindex="....."></use_agent_install_guide>,
Script : AgentInstallGuideZIndex
XML : <use_agent_install_guide logo_url="....."></use_agent_install_guide>,
Script : AgentInstallGuideLogoUrl

Script 설정이 우선 적용됩니다.
Agent 설치시 설치 안내창 노출여부를 설정합니다.

remarks

use_agent_install_guide  기본값이 "1" Agent 설치시 설치 안내창 노출이고, "0"은 노출하지 않습니다.
zindex  Agent 설치 안내창에 대한 ZIndex를 설정합니다. (기본값: 1000)
logo_url  Agent 설치 안내창의 로고 이미지 url을 설정합니다. (기본값: "images/BI-raonk.png")

sample code

Agent 설치시 설치 안내창 노출여부를 노출로 설정합니다.
Agent 설치 안내창의 ZIndex는 1500으로 설정합니다.
Agent 설치 안내창의 로고 이미지 url을 설정합니다.
