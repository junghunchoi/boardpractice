K Upload :: UseFormDataInRangeInfo

XML : <use_form_data_in_range_info>.....</use_form_data_in_range_info>,
Script : UseFormDataInRangeInfo

Script 설정이 우선 적용됩니다.
formdata에 range 정보의 추가 여부를 설정합니다.

remarks

기본값은 "1" 이며 설정 시 formdata에 range 정보를 추가 하고,
"0" 으로 설정 시 formdata에 range 정보를 추가하지 않도록 설정합니다.

sample code

formdata에 range 정보를 추가하지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <use_form_data_in_range_info>0</use_form_data_in_range_info>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            UseFormDataInRangeInfo: "0"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
