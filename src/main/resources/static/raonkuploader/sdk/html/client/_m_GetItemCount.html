K Upload :: GetItemCount

int GetItemCount(uploadID)

업로드 파일리스트에 있는 전체 파일의 수를 가져옵니다.

return value

int  업로드 파일리스트에 있는 전체 파일의 수를 의미합니다.

parameters

uploadID  업로드의 id를 의미합니다.

mode  파일의 종류를 구분할 값을 의미합니다.
         total : 전체 파일
         new : 신규 파일
         web : WEB 파일
         delete : WEB 파일 중 삭제된 파일(전체파일 개수에는 포함되지 않음)

remarks

없음.

<script type="text/javascript">

            var count = RAONKUPLOAD.GetItemCount("upload1", "total");
        }
