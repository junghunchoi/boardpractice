K Upload :: ThumbsViewMode

XML : <thumbs_view_mode>.....</thumbs_view_mode>, Script : ThumbsViewMode

Script 설정이 우선 적용됩니다.
썸네일 보기 모드(업로드의 views 모드가 thumbs)시 적용됩니다.
썸네일 모드시 표시 내용을 설정합니다.

remarks

"0" 으로 설정 시 썸네일, 파일명, 크기를 모두 표시하고,
"1" 로 설정 시 썸네일만 표시하고,
"2" 로 설정 시 썸네일, 파일명을 표시하고,
"3" 으로 설정 시 썸네일, 크기를 표시합니다.

sample code

썸네일 보기 모드에서 썸네일만 표시하도록 설정합니다.

XML

.....
<setting>
    .....
    <thumbs_view_mode>1</thumbs_view_mode>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            ThumbsViewMode: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
