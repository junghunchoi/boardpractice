K Upload :: SetCloudInfo

void SetCloudInfo(infoObj, uploadID)

클라우드 업로드 정보를 설정합니다.

return value

없음.

parameters

infoObj  클라우드 정보를 담은 오브젝트를 의미합니다.
- infoObj.Type  클라우드의 종류를 의미합니다.
- infoObj.AuthMethod  클라우드의 인증방식을 의미합니다.
- infoObj.AccessKeyID  클라우드의 실제 AccessKey를 의미합니다.
- infoObj.SecretKey  클라우드의 실제 SecretKey를 의미합니다.
- infoObj.EndPoint  클라우드의 실제 EndPoint 주소를 의미합니다.
- infoObj.Bucket  클라우드의 실제 Bucket을 의미합니다.
- infoObj.AuthHeader  자격부여 헤더를 의미합니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.

                Bucket : ".....",
                AuthHeader : "....."

        }
</script>
