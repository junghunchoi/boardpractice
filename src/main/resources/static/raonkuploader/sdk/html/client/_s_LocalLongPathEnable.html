K Upload :: LocalLongPathEnable

XML : <local_long_path_enable>.....</local_long_path_enable>, Script : LocalLongPathEnable

Script 설정이 우선 적용됩니다.
다운로드 시 다운로드 경로가 긴 경우를 허용 가능 여부를 설정합니다.

remarks

Win32 환경에서 설정이 적용됩니다.
기본값은 "0" 이며 다운로드 경로가 긴 경우를 허용하지 않고,
"1"로 설정 시 다운로드 경로가 긴 경우를 허용 하도록 설정합니다.

sample code

다운로드 시 다운로드 경로가 긴 경우를 허용 가능하도록 설정합니다.

XML

.....
<setting>
    .....
    <local_long_path_enable>1</local_long_path_enable>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            LocalLongPathEnable: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
