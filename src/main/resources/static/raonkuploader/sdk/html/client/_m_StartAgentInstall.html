[Agent 전용]

K Upload :: StartAgentInstall

void StartAgentInstall(uploadID)

Agent 설치 팝업을 띄우고 설치파일 다운로드를 수행합니다.

return value

없음.

parameters

uploadID  업로드의 id를 의미합니다.

remarks

웹표준모드에서도 해당 API가 동작은 하나, 설치파일을 설치하여도 runtime을 변경하지 않는 한 업로드가 Agent모드로 동작하지는 않습니다.
웹표준모드에서 Agent 설치 후 runtime 변경 시 별도 설치없이 이미 설치된 Agent로 Agent 모드가 동작합니다.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function StartAgentInstall() {
            RAONKUPLOAD.StartAgentInstall("upload1");
        }
</script>
