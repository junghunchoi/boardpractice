K Upload :: GetConfig

string GetConfig(configName, uploadID)

업로드에 해당 이름으로 설정된 값을 가져옵니다.

return value

string  설정된 config 값 입니다.

parameters

configName  가져올 설정명을 의미합니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
<script type="text/javascript">
        function GetConfig() {
            var conf = RAONKUPLOAD.GetConfig(".....", "upload1");
        }
</script>

        </script>
