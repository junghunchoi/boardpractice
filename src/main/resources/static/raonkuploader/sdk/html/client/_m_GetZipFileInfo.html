K Upload :: GetZipFileInfo

void GetZipFileInfo(options, uploadID)

해당 경로에 해당하는 zip 파일 안의 파일 리스트 정보를 callbackFunc의 파라미터로 반환받습니다.

return value

callBackFunc의 첫번째 파라미터에 return값을 반환받습니다.
object   zip 파일 안의 파일 리스트 정보를 포함한 의미합니다..
- object.data  압축파일내에 존재하는 파일의 정보를 포함한 Array를 의미합니다.
- object.errorMessage  해당 API동작에 실패하였을 경우 errorMessage를 의미합니다..
- object.success  해당 API동작의 성공여부를 의미합니다.

parameters

options  API 실행을 위한 정보를 담고있는 object를 의미합니다.
- path  압축 파일의 물리적 경로를 의미합니다.
- callback  수행할 callBackFunc를 의미합니다.
uploadID  업로드의 id를 의미합니다.

remarks

없음.
