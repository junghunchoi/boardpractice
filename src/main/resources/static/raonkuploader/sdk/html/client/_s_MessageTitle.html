K Upload :: MessageTitle

XML : <message_title>.....</message_title>, Script : MessageTitle

Script 설정이 우선 적용됩니다.
메시지 창의 타이틀을 설정합니다.

remarks

기본값은 "RAON K" 이며, 메세지 창의 타이틀로 설정할 문구를 입력합니다.

sample code
