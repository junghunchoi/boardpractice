K Upload :: SaveAndOpenMulti

XML : <save_and_open_multi>.....</save_and_open_multi>, Script : SaveAndOpenMulti

Script 설정이 우선 적용됩니다.
파일 저장 후 열기 시 여러 파일을 저장 후 열기 가능하도록 또는 한 파일만 저장 후 열기 가능하도록 설정합니다.

remarks

기본값은 "0" 이며 설정 시 파일 저장 후 열기시 한 파일만 저장 후 열기하도록 설정하고,
"1" 으로 설정 시 파일 저장 후 열기시 여러 파일을 저장 후 열기 가능하도록 설정합니다.

sample code

파일 저장 후 열기시 여러 파일을 저장 후 열기 가능하도록 설정합니다.

XML

.....
<uploader_setting>
    .....
    <save_and_open_multi>1</save_and_open_multi>
    .....
</uploader_setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            SaveAndOpenMulti: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
