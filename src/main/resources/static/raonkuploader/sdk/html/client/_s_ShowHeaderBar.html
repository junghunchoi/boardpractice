K Upload :: ShowHeaderBar

XML : <show_header_bar>.....</show_header_bar>, Script : HeaderBar
XML : <show_header_bar header_height="....."></show_header_bar>, Script : CustomHeaderHeight

Script 설정이 우선 적용됩니다.
업로드의 상단을 표시하거나 숨기는 기능을 설정합니다.

remarks

show_header_bar  기본값은 "1" 이고, "0" 으로 설정시 상단을 표시하지 않습니다.
header_height  최소값은 "28" 이며 업로드 상단 HeaderBar의 높이(height + padding)를 설정합니다.

sample code

업로드의 상단을 표시하도록 설정합니다.
업로드의 상단 HeaderBar의 높이를 40px(height + padding)로 설정합니다.

XML

.....
<setting>
    .....
    <show_header_bar header_height="40">1</show_header_bar>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            HeaderBar: '1',
            CustomHeaderHeight: '40'
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
