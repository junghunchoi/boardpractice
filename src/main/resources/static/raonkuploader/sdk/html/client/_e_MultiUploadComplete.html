K Upload :: MultiUploadComplete

boolean RAONKUPLOAD_MultiUploadComplete(uploadID)

MultiTransfer API 사용 후 완료 시 발생합니다.

return value

없음.

parameters

array  멀티 업로드 정보(status, uploadID)가 있는 object들을 포함한 array 입니다.

remarks

모든 업로드가 완료된 후 1회 발생합니다.

    function RAONKUPLOAD_MultiUploadComplete(uploadID) {
        // MultiTransfer API 사용 후 완료 시 처리내용
    }
</script>
