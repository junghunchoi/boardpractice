K Upload :: DEXT5Install

XML : <DEXT5_install>.....</DEXT5_install>, Script : DEXT5Install

Script 설정이 우선 적용됩니다.
Agent 모드 사용시 DEXT5 플러그인의 설치여부를 설정합니다.

remarks

"0" 으로 설정시 DEXT5 플러그인 설치 하지 않도록 설정하고,
"1" 로 설정시 DEXT5 플러그인도 설치 하도록 설정합니다.

sample code

Agent 모드 사용시 DEXT5 플러그인도 설치하도록 설정합니다.

XML

.....
<setting>
    .....
    <DEXT5_install>1</DEXT5_install>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var raonkParam = {
            .....
            DEXT5Install: "1"
        }

        new RAONKUpload(raonkParam);
    </script>
</div>
