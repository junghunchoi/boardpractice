[Agent 전용]

K Upload :: EnableAgentShortcut

XML : <enable_agent_shortcut>.....</enable_agent_shortcut>, Script : EnableAgentShortcut

Script 설정이 우선 적용됩니다.
Agent 단축키 기능의 활성화 여부를 설정합니다.

remarks

"0" 으로 설정시 Agent 단축키 기능을 비활성화 하고,
"1" 로 설정시 Agent 단축키 기능을 활성화 합니다.

sample code

Agent 단축키 기능을 비활성화 합니다.

XML

.....
<setting>
    .....
    <enable_agent_shortcut>0</enable_agent_shortcut>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
