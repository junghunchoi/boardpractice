K Upload :: CheckNetworkSpeed

XML : <check_network_speed>.....</check_network_speed>, Script : CheckNetworkSpeed

Script 설정이 우선 적용됩니다.
네트워크 속도체크 여부를 설정합니다.

remarks

"0" 으로 설정시 네트워크 속도체크 하지 않도록 설정하고,
"1" 로 설정시 네트워크 속도체크 하도록 설정합니다.

sample code

네트워크 속도체크 하도록 설정합니다.

XML

.....
<setting>
    .....
    <check_network_speed>1</check_network_speed>
    .....
</setting>
.....

Script

<script type="text/javascript" src="raonkupload/js/raonkupload.js"></script>
	
<!-- ..... 생략 ..... -->

    <script type="text/javascript">
        var raonkParam = {

        }

    </script>
</div>
